# 安尼尔 P0 药酒交易时间线修复计划

## 问题诊断

当前存在死循环：

```
猫女苏醒需要魔法药酒（Day02数据层定义）
    → 药酒来自安尼尔 Stage 3 交易
    → 但 Stage 3 触发条件写的是"猫女少女苏醒后第3天"
    → 猫还没醒 → 没有少女 → 触发不了 Stage 3 → 拿不到药酒
    → 猫永远醒不了
```

## 修复方案

### 魔法药酒提前到 Stage 2，苏醒放在 Stage 2→3 间隙

| Stage | 安尼尔事件 | 老麦态度 | 猫女状态 |
|-------|-----------|----------|----------|
| **1** (Day02) | 初来，安静喝酒看图纸 | 抗拒（像前上司） | 紫猫在窗台观察 |
| **2** | 吐槽之夜 + **药酒交易**（安尼尔把药酒给老麦） | 抗拒→还行 | 猫逐渐亲近酒馆 |
| **2→3 间隙** | — | 老麦配出魔法药酒 | **猫偷喝 → 紫光大作 → Lv.3 少女灵体首次显现** |
| **3** | 讲故事（学院劝退→倒卖药剂→总负责人）+ 见到猫女少女 | 还行→不满（听到给工人用药的细节后沉默） | 已苏醒，少女形态出现在酒馆 |
| **4** | 结局分支。**检查：sylvena_level >= 3** | 视结局而定 | 已苏醒 → 结局3「苗」可用 |

### 关键改动点

1. **药酒交易从 Stage 3 搬到 Stage 2。** 安尼尔在吐槽之夜把药酒给了老麦。药酒来源的细节（他为什么随身带着、是样品还是特意带来的）暂时留空，后续再补——先把结构搭对。

2. **猫女苏醒放在 Stage 2→3 间隙。** 老麦用安尼尔给的药酒配出魔法药酒，猫偷喝后 Lv.3 少女灵体首次显现。到 Stage 3 时猫女已经是醒着的状态。

3. **Stage 3 重新聚焦。** 不再是"交易"，而是安尼尔讲述完整人生经历 + 第一次见到猫女少女（学院袍子触发他的回忆）。老麦听到给工人用药的细节后态度从"还行"转为"不满"——这是 Stage 3 的情绪主线。

4. **Stage 4 结局3「苗」触发条件下调。** 从"猫女已回学院任教（客座教授结局）"→ 改为"猫女已苏醒（sylvena_level >= 3，即少女灵体状态）"。安尼尔 Stage 4 时猫女必然已醒，结局3「苗」的触发条件始终满足。

5. **引擎侧新增固定时间节点检查。** Stage 4 开始时检查 `sylvena_level >= 3`，决定结局3「苗」是否可用。

### 已知留空（后续补）

- Stage 2 中安尼尔把药酒给老麦的具体理由（随身携带？特意带来？）——暂不解释，优先保证时间线通畅。

### 不改动的部分

- 结局A「破土」、结局B「永眠的花」逻辑不变
- 药酒交易的对话内容（老麦提议以药换酒、安尼尔同意）保留，只是从 Stage 3 搬到 Stage 2
- 调酒永远是啤酒不变
- 安尼尔讲述人生经历的内容保留，留在 Stage 3

### 对猫女线的影响

- 猫女 Lv.3 苏醒时间点固定在"安尼尔 Stage 2 之后、Stage 3 之前"
- 猫女 Lv.4-7 成长在安尼尔 Stage 3 之后继续推进
- 安尼尔 Stage 3 时猫女已是少女形态，两人可以有直接互动
