## Day 1 · 警探 Lendro 第1次来访 ✅

**来源**：复用 `脚本/Day01_警探_第1次.md`（完整保留）

### 场景设置

```
[BG tavern_night]         1920s 美国酒馆，深色木板，暖黄暗灯
[BGM jazz_piano_low]     低沉爵士钢琴
[LIGHT 暖黄偏暗，吧台灯亮，角落阴影重]
```

### 叙事目的

建立游戏的核心循环（调酒+倾听），同时埋下两条线：
1. 警探不是来查案的——是用盘查当借口的
2. 窗外有东西（猫的伏笔）

### 情感节拍

```
1. 紧张：警探以"盘查"入场，气氛压抑
2. 松弛：白酒让伪装裂开一道缝——"没兑水。也没兑劣质糖浆。"
3. 好奇：窗外黑影飞掠（猫伏笔）
4. 荒诞：邪教徒闯入，"深渊？码头发臭的排水沟也叫深渊？"
5. 紧张→信任：诡异氛围中，警探确认酒保不是瞎子。"老骨头，明天最好还亮着灯。"
```

### 关键选择

| 节点 | 选项 | 最佳路径 | 影响 |
|------|------|---------|------|
| ch_01_01 回应盘查 | A:超然 / B:直白 / C:反问 | C | C: trust+3, deflection-2 |
| drink_01_01 | 白酒/威士忌/啤酒/柠檬水 | **白酒** | trust+5, deflection-5 |
| ch_01_02 工业区 | A:递烟 / B:承认 / C:沉默 | A(递烟) | trust+4 |
| ch_01_03 邪教徒 | A:"烟灰要掉了" / B:冰水镇场 / C:让警探处理 | A | 最自然的配合 |
| ch_01_04 临别 | A:危险 / B:扔掉 / C:倒酒压惊 | 各有风味 | trust 不同程度变化 |

### 情感节拍

```

5. 虚无：玩弄带有流星刻印的打火机。讲述太空的绝对安静带来的茫然。
6. 化解：老麦重新倒满酒/擦灰尘/开玩笑，用踏实的生活动作化解了虚无感。
7. 离去：喝完酒离开，不需要安慰。在门边低语："我活着就是一切。"
```

## Day 11 · 哲学神 · 终曲 🆕 脚本6

**来源**：全新编写

### 前置条件

```
Day 10 完成（自动解锁）
```

### 场景设置

```
[BG tavern_night_empty]           打烊后的空酒馆。纯白墙+吧台+酒架。
[BGM absolute_silence]            完全静默。没有音乐。只有吧台的木头偶尔吱一声。
[PROP 酒馆里到处都是痕迹：吧台上的弹孔、墙上的图纸（如果维修师留下的）、
      酒架旁的魔法杖、窗台上的蒲公英。
      猫在窗台上——安静地。帽子没有歪。吊坠不亮了。它在等。]
```

### 叙事目的

全游戏收束。不是高潮——是余韵。让玩家坐五分钟，看着这个来过很多人的地方，然后一个人走进来，摸过所有的痕迹，说一句话，然后消失。

### 情感节拍

```
1. 门开了：
        没有风铃。没有脚步。
        一个普通的男人站在门口。不属于任何时代。不引人注目。
        他没有进来——他走过来了。像这儿本来就是他该在的地方。
2. 修复：
        走过猫的身边。伸手——轻轻抚过猫的耳朵。拿掉沾在上面的蒲公英。
        猫没躲。甚至没动。它眯起眼睛——像是等了很久。
        
        走过吧台。擦过维修师的图纸——抹掉一根多余的线头。
        戳了戳墙上的弹孔——墙壁修补如新。
        
        动作安静、熟练。像做过无数次。
3. 恢复：
        他回到吧台前。坐下。
        然后——一切开始恢复原样。
        
        猫在洗脸。蒲公英又飘回来，沾在猫的耳朵上。
        图纸变旧。线头回到原位。
        弹头飞来，重新嵌入弹孔。
        
        他不修了。就看着这些"错误"回来。
4. 痕迹（无声的认出）：
        他一样一样摸过那些人留在酒馆里的东西。
        
        → 弹孔（警探）：你选择了让过去存在。我也可以让过去不存在。但你让它留在了墙上。
        → 怀表碎片（赌徒）：你挣扎了一辈子。这挣扎本身——就是我不再是"我"的意义。
        → 倒扣的酒杯（死神）：没有期限的东西，连"期限"这个概念都没有。而你，把酒杯倒扣在吧台上。
        → 蒲公英（猫女）：永恒不是错误。错误的是你以为永恒能给你答案。但这朵蒲公英——你留在这里了。
        → 锚点坐标/空位（牛仔）：在无限里刻一个坐标。我也面对过无限。所以我造了墙。
        → 图纸（维修师）：画了二十年。我下山后的世界，就该有人画这样的图。

        这些想法不对任何人说。只是场景写作的参照。
        
5. 唯一一句台词：
        他停留在蒲公英前。蒲公英在他指间轻轻转动。
        
        "这就是我想要的。"
        "他们来过，而你记得。"
        
        蒲公英从他指间消散。
        他的轮廓随之淡去。
        
        不是消失——是退场。像一个演员知道自己的戏份结束了。
6. 尾声：
        酒保看着空了的座位。停顿了很久。
        然后——倒了一杯酒。放在那个空位前。像留给一个会回来的人。
        
        拿起杯子，开始擦。
        猫从窗台跳下来。走到那个空位。闻了闻。
        灯——保持亮着。
        
        【END OF DEMO】
```
