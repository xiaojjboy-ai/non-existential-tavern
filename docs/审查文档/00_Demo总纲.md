# 非存在主义酒馆 · Demo 阶段核心设计纲要

---

## 一、项目愿景与核心定位

一个类《VA-11 Hall-A》的文字冒险 Demo。玩家扮演酒保老麦，在时空之外的酒馆里给形形色色的客人调酒、倾听他们的故事。Demo 时长约 90 分钟，覆盖 7 个角色、11 个夜晚。

完整版设计有 10 个角色和双层叙事反转，但 Demo 版本的核心目标只做第一层——通过极致的文本质感和沉浸交互，建立玩家与角色、酒馆之间的深层情感链接。

---

## 二、极简核心交互系统 (Minimalist Interaction)

```
客人入场 → 对话（3-4个选择节点）→ 调酒判定 → 对话深入 → 客人离场 → 打烊机制 → 下一日
```

本作剔除了传统的资源管理、探索与金钱系统。将所有的游玩体验收束于吧台前的文字与动作。

### 玩家核心操作链

| 操作 | 触发机制 | 叙事与数值影响 |
|------|---------|------|
| **选项交互** | 对话关键节点（每场 3-4 次） | 改变客人态度、决定好感度流向、影响信息揭示深度 |
| **调酒判定** | 每场剧情中段强制触发一次 | 调酒正确：大幅提升信任（Trust）；调酒错误：遭遇客人言语嫌弃，但不造成 Game Over |
| **打烊投喂** | 每天打烊前的可选操作 | 积累隐性变量（猫女亲和度），为特定灵体事件（Day 8/10）提供解锁条件 |

## 三、叙事基调与绝对红线

### 整体氛围
**"深夜、吧台、一个愿意听的人。"**
这是一部聚焦于“成年人的安静崩溃与自我消化”的作品。角色来到酒馆并非为了生理上的解渴，而是因为他们在广袤宇宙或残酷现实中，已经找不到一个愿意倾听的人。

### 核心写作原则（必须严格遵守）
1. **老麦的交互行为学**：老麦绝对不说教、不拯救。玩家在扮演老麦时的选项，必须是具体的动作（擦杯子、推杯子）、生活化刻度（多倒半指酒）或蓝领式的干涩调侃，严禁出现“我觉得你很痛苦”等心理分析式选项。
2. **拒绝廉价煽情**：最沉重的伤疤必须用最轻盈的口吻揭开。
3. **行动定义弧光**：角色的觉醒与释然不依赖长篇独白，而是依赖关键动作（如赌徒主动走向死神碰杯并砸碎怀表）。
4. **克制的留白与生理性隐喻**：绝不允许角色进行自我精神分析或挂着心理学术语。所有的创伤必须通过生理反应（对特定声音的过敏、肌肉的紧绷、必须睁眼的失眠）来具象化呈现。

### 角色调性锚点

| 角色 | 调性基准 | 核心意象台词 |
|------|------|--------|
| 警探 Lendro | 工作经验总结出的毒辣侧写与话痨，掩盖对安静的恐惧，天马行空必落于实处 | "我还在查。查到死为止。" |
| 维修师 安尼尔 | 疲惫但坚韧的理想主义，油腻机油下的绝对纯粹 | "啤酒。永远是啤酒。" |
| 牛仔 | 干练洒脱，用随遇而安掩饰太空迷失的茫然与沧桑 | "我活着就是一切。" |
| 赌徒 Aletor | 从极度高压下的神经质，到防线松动后的自我确认 | 碰杯那一下，什么都不用说 |
| 死神 Lethus | 优雅而绝对的中立，死亡是无差别的艺术呈现 | "墙不会移动。墙只是在那里。" |
| 猫女 Sylvena | 从初醒断裂的怯懦灵体，到恢复全盛记忆的傲慢天才与深刻委屈 | "这不公平。天才不应该被困在猫身体里。" |
| 哲学神 | 剥离全能神性，回归人性感官去体会辛辣与裂痕的中年男人 | "原来这杯酒，喝下去是这种味道。" |

| Day | 登场角色 | 核心事件抽象 | 情绪底色 |
|-----|------|---------|------|
| 1 | 警探 第1次 | 神经质盘查→白酒→锐利识破邪教徒，建立庇护所认知 | 紧绷的试探 |

| 6 | 赌徒 第2次 | 放松防线主动要烈酒，讲出最令自己骄傲的逃亡博弈 | 短暂的自我确认 |

## 五、技术边界与系统规范

本部分约束研发实现的边界，严禁过度设计。

1. **状态机驱动**：严格遵循 Markdown + YAML 混合编译逻辑，一切分支跳转和调酒判定基于节点推进。
2. **动态选项树**：选项影响亲和度（affinity）或防御值（defense）等状态变量，系统需支持跨日持久化存储，并可作为后续剧情的解锁阀值。
3. **极简表现层**：不支持多角色同时在场的复杂演出交互。Day 9 死神为叠加特效层，Day 8/10 猫女灵体仅作为窗台位置的定点视觉变化，大幅降低实现负荷。
4. **剔除冗余系统**：不引入金钱核算、不需要二周目框架、剔除时间流逝判定。
