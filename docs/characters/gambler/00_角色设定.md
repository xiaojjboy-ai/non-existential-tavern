# 赌徒 Aletor — Character Canon

## Core Identity

一个眼眶发黑、极度亢奋又极度绝望的“亡命徒”。身上混杂着废土泥土、陈旧血迹和昂贵魔法布料的气息。他频繁地看怀表，手里死死攥着一根紫色的传送法杖。
他看起来像一个疯狂的赌徒，实际上，他曾是这个世界上最伟大的探险家。

**核心命题**：为了生存而奔跑，直到生存本身变成最深沉的恐惧。

## Immutable Canon

- **曾经的荣光**：他曾是世界上最伟大的探险家。去过无数极端绝境，帮助魔法学院到极度危险的地带取过东西。他探索过时间的缝隙和世界的边缘。
- **两大奇物**：
  - **时间之峡的怀表**：他在时间之峡偶然获得的奇物，能实现短暂的“时间暂停”。
  - **紫色传送法杖**：这是他用命在魔法学院换来的东西。这是天才魔女 Sylvena 年轻时的毕业作品，带有时间和次数的随机传送限制。他利用怀表的“时停”机制，卡了法杖的 Bug，跳过了这些限制。
- **陨落与死神的凝视**：因为在死亡边缘徘徊了太多次，他像是不断在“规则的高墙”上翻越，最终引来了死神（Lethus）的凝视。
- **时停中的赌约**：在一次跌下山崖的致命危机中，他习惯性地按下了怀表时停。但在那冻结的一瞬间，死神已经站在了他面前。因为规则的冲撞（处于时停状态），死神无法当场带走他。死神皮笑肉不笑地定下赌约：“规则就是规则，这次我依然会放过你。但是你的运气已经用完了。从今天开始，我将不停地追逐你，直到你的死期到来。好处就是，在我没有追到你之前，这一天还不是你的死期。”
- **恐惧的诞生**：在时停的瞬间，他感受到了死神毫无生机的寒气。他所有的求生手段、探险经验在绝对死亡面前成了笑话。强烈的求生欲压倒了一切，伟大的探险家死了，一个为了逃离死神而疯狂使用法杖的“赌徒”诞生了。

## Core Wound / False Belief / True Need

- `core_wound`：时停瞬间死神贴脸时的极寒感。那是绝对力量对凡人技巧的降维碾压。
- `false_belief`：“只要我跑得够快、传送得够随机，只要我一直赢下去，今天就不会是我的死期。”
- `true_need`：停下来。接受死亡的必然性，找回曾经作为“探险家”面对未知的勇气，而不是作为“猎物”逃亡。
- `core_question`：当生存本身变成了无止境的逃亡时，多活的每一秒是奖赏，还是折磨？

## Stage List

赌徒的 Stage 是与“死神逼近”的时间线严格绑定的。

| Stage | 状态 | 标志性变化 |
|---|---|---|
| 1-1 | 亢奋、神经质（Day 05） | 刚赢了死神一次（成功传送逃脱）。极度兴奋，大口喝酒，频繁看表。手里死死攥着法杖。 |
| 1-2 | 焦躁、强装镇定（Day 10） | 输了一半。黑眼圈加重，传送法杖的魔力开始不稳定。要求喝最烈的酒，不断看门外。 |
| 2-1 | 崩溃、发抖（Day 13 擦肩而过） | 彻底崩溃，躲在酒馆角落发抖。试图用买醉来逃避现实。死神在一小时后就会到达。 |
| 3 | 终局对决（Day Y） | 放弃逃跑，把法杖留在吧台。在酒馆里平静地直面推门而入的死神。 |

## Cross-character Links

- **死神 Lethus（追捕者）**：绝对的猎手。死神不带恶意，只是机械地执行那个“赌约”。死神的从容与赌徒的癫狂形成极致的对比。
- **猫女 Sylvena（法杖的制造者）**：她制造了法杖，他用这根法杖卡 Bug 逃命。两人构成了绝妙的镜像——赌徒用**永远的逃跑**来反抗命运（死神），猫女用**绝对的静止**来反抗命运（肉体的衰老）。两个试图颠覆规则的人，最终都被规则囚禁。
- **老麦（酒保）**：在 Day 13，老麦不动声色地擦去了赌徒留在吧台的冰水渍，面对死神保持了沉默，给了赌徒最后一次喘息的机会。
- **维修师 En Neer**：两人在技术理念上完全对立。维修师追求“确定性”（轨道传送阵），而赌徒把一切押在“随机性”（随机传送法杖）上。

## Forbidden Canon Drift

- 不得把赌徒写成一个真的因为“赌钱”而落魄的人。他的“赌”是跟死神赌命。
- 不得让他展现出懦弱的底层混混气质。他骨子里曾经是**最伟大的探险家**，他的崩溃是因为见识到了超越凡人认知的终极恐惧。
- 绝不能让他离开法杖和怀表。这是他能活到现在的唯一依仗。
- 不得在终局前让他展现出“视死如归”。他极度渴望活着，求生欲是他唯一的驱动力。
