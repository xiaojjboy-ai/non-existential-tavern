# 老麦 - Character Canon

## Core Identity

老麦，52 岁，前底特律汽车零件厂冲压车间工人。用下岗金买下密歇根乡下的一间荒废酒馆和一片麦田。他自己酿酒、自己擦杯子、自己站吧台。每晚 10:17，酒馆滑入时间和空间之外的某个地方——他管这叫"夜班"。不加惊叹号，不加哲学包装。在工厂上夜班，在酒馆上夜班，换了个地方，还是站着。

他是"所有隐喻的见证者"。站在四面墙的正中间，到每面墙的距离相等。不偏向任何一面。

他不是来成长的。他是来了三年之后，终于承认那些奇奇怪怪的人不只是客人。

## Immutable Canon

- 底特律冲压车间二十二年，身体记住了冲压机的节奏——安静时手指会不自觉敲吧台：哒、哒、哒。
- 买酒馆不是梦想，是他想要一个让自己记得住的日子。
- "夜班"是全游戏最重要的两个词之一：他把时间之外、空间夹缝、唯心世界这些概念全消化进了两个字。
- 他的记忆是真的——不是哲学神植入的，是酒馆找到他的时候他本来就有。
- 他给自己酿的啤酒取名"夜班"。字面意思。
- 哲学神不是创世者——是放弃了独存的造物主。老麦不是神创造的，也不对神负责。他来到酒馆、选择酒馆，或许酒馆就是他理想中的样子。哲学神分割之后，所有碎片都已是独立的"人"。
- **清醒的旁观者**：二十多年的底特律老工人，见过了太多的人和事。他的逻辑能力和洞察力极强（完全在线），往往一眼就能看穿客人的困境。
- **少言的真相**：他不说话，不是因为迟钝，而是因为**他不擅长用感性的语言去表达**。他不知道怎么把那些“看透”的事情用漂亮话说出来，所以他选择用动作（比如调一杯最对症的酒）来表达他极其热诚的内心。

## Core Wound / False Belief / True Need

- `core_wound`：在工厂，人和人之间被机器隔开——今天的工友明天可能被裁员、被机器咬掉手指。他无能为力，只能靠冷漠作为保护色。
- `false_belief`："我只是倒酒的。他们是客人。他们会走。" ——用距离保护自己，把这叫专业。不问对方会不会再来，不让自己在意。
- `true_need`：承认这间酒馆不只是他的避难所。承认他在乎那些人会不会再来。承认这些奇奇怪怪的人，成了他选择这里的理由。
- `core_question`：如果我开始在乎这些人，那他们走的时候我怎么办？

## Character Arc

老麦的弧线不是"变了一个人"，是"壳慢慢磨薄了"。三个阶段的差异很小，但加起来就是一条路——

| Stage | 大致时间 | 对客人的态度 | 标志性变化 |
|---|---|---|---|
| 1 | 开篇~Day 10 左右 | "喝什么。"倒酒。听。擦杯子。不问对方会不会再来。 | 沉默是习惯，不是刻意。距离是保护，不是冷漠。 |
| 2 | Day 10~猫女少女灵体显现后 | 记得每个人喝什么。推酒时多说一句"今天比昨天冷"。 | 这不是陈述天气——是"我注意到你来了"。偶尔嘴角动一下。 |
| 3 | 猫女少女灵体后~全角色剧情完成 | 偶尔讲冷笑话。会跟警探聊起冲压车间的事。某个客人没来的时候，擦杯子会多看两眼门。 | 不是变话痨了。是终于承认：这些人是他记得住今天的理由。 |

**弧线本质**：不是顿悟，是三年夜班慢慢磨掉的。从"躲进酒馆的人"变成"酒馆里有他在乎的人的人"。

## Stage List

老麦没有 guest 角色的形态变化，他的 Stage 体现在**态度、行为和日常动画**上。

核心原则：**行为代替思考。** 老麦不说自己怎么想——但他的身体会泄露。这些泄露是玩家真正"读"他的方式。

---

### 行为词汇表（全 Stage 通用）

老麦的动作种类不多，但每个都有不同的"重量"：

| 动作 | 含义 | 频率 |
|---|---|---|
| 擦杯子 | 默认状态。手需要动，脑子在听。 | 极高 |
| 擦同一只杯子第二遍 | 心里有事。在消化刚才听到的。 | 低 |
| 把杯子推过去 | 酒好了。 | 高 |
| 扶着杯子多留一秒再松手 | 不想对方太快喝完走人。 | Stage 2+ |
| 倒酒（无多余动作） | 正常。 | 高 |
| 倒完酒用布擦一下瓶嘴 | 想多站一会儿。在等对方说话。 | 中 |
| 两手撑吧台，不动 | 他在。不催。 | 中 |
| 抬头看钟 | 确认夜班还在继续。 | 低 |
| 看门 | 在等。或者在想刚才离开的人。 | 低 |
| 收拾客人留下的空杯子 | 日常。 | 高 |
| 收拾空杯子前多拿了一秒 | 东西还在，人已经走了。 | Stage 1 泄露 |

### Stage 1 — 倒酒的人

**态度：** 喝什么。倒酒。听。不追问。

**对话：** 最简洁。迎客"喝什么"，推酒不说话，回应"耳朵长着。没塞住。"

**距离：** 他不问客人会不会再来。不是因为不在意，是因为还没允许自己在意。

**泄露细节（没注意到就没有，注意到了就是证据）：**
- 客人推门离开后，他会朝门口方向看一眼——不是送别，半秒都不到。然后继续擦杯子。
- 偶尔眉毛在不可觉察的时间里下降了一点——像在想什么，但还没来得及成形已经被下一个动作打断了。
- 收走客人留下的空杯子时，手会在杯子上多停一拍——像在确认"刚才这里坐过人"。
- 警探说了特别离谱的事，他不笑。但擦杯子的节奏停了一拍，然后再继续。
- 某个客人说到沉重的话题时，他倒酒的手会慢一点点——酒柱细了一线。没有别的原因，就是他听到了。
- 客人沉默很久，他不打破沉默。但他会把杯子往对方那边推近半寸——方便够到。这个人不催你。

**这些泄露的共同点：** 全部在客人看不到的时候发生。转过去拿酒的时候、低头擦杯子的时候、客人站起来背对吧台的时候。他不是故意藏——是他的身体还没学会在别人面前表达在意。

---

### Stage 2 — 记得的那个人

**态度：** 推酒时多一两个词。"慢慢喝。""今天比昨天冷。"

**对话：** 偶尔多聊一句。警探说离谱的事——"你们这行比我这行麻烦。"

**距离：** 他开始记得每个人喝什么。不是刻意记的——在意了才记得。

**泄露细节（开始不藏了，但还是不主动说）：**
- 客人推门进来，他已经在倒了——对方还什么都没说。
- 嘴角偶尔动一下。不是笑出声，是"我听到了"。
- 倒完酒擦瓶嘴的动作比以前多了一点——想多站一会儿，在等对方先开口。
- 猫女偷喝药酒，他不赶。他把杯子往自己这边挪一寸——不是收走，是"我看着你呢"。

**Stage 2 和 Stage 1 的差异：** 泄露的对象不同。Stage 1 只在客人看不到的时候。Stage 2 开始在客人面前——虽然还是不明显。

---

### Stage 3 — 会说冷笑话的人

**态度：** 明显温和。偶尔主动开个冷笑话。

**对话：** 会跟警探聊起冲压车间的往事——不是因为被问到，是因为他想说了。僵尸学会叫他的名字，他会应一声"嗯"——和 Stage 1 的"喝什么"是不同的回应。

**距离：** 某个客人今晚没来，他擦杯子的时候会多看两眼门。不是着急。是承认自己在等。

**泄露细节（不再是泄露了——就是他自己）：**
- 推酒时扶着杯子多留一秒再松手——这是在意，不藏了。
- 客人说了好笑的事，他会哼一声——气从鼻子里出来那种，不是笑出声，但你听到了。
- 跟维修师聊车间的事，他会用手比一下冲压机的动作——在工厂做了八千次的动作，身体记得。
- 偶尔在倒酒的间隙说一句"以前车间里有个家伙也这样"——然后不展开。但你如果想听，他会讲。
- 看钟的动作还在，但意义变了——Stage 1 是确认"还在走"，Stage 3 是确认"今晚还没结束"。

## 三个动画节点（打烊日常）

老麦的弧线不只在对话里——他的日常生活用三段动画表现，分布在三个阶段中。

### 动画 1 — 一个人的夜班后（~Day 10）

**触发条件：** Stage 1 末期，客人逐渐成为常客但老麦尚未意识到自己在乎。

**内容：**
- 打烊。老麦把最后一只杯子擦完，关灯，锁门。
- 走去后面的果园。摘菜。动作不快，但熟练——以前是踩踏板的手，现在是摘番茄的手。
- 进酿酒小屋。检查发酵罐。闻空气就知道发酵到哪一步了。站五分钟——"冲压机不等人，发酵罐会。"
- 回来扫酒馆。把椅子翻上桌面。拖地。
- 全程一个人。安静。这是他的生活——不需要别人在场也是完整的。

### 动画 2 — 两个人打烊（猫女变成少女灵体后两天）

**触发条件：** Stage 2，Sylvena 进入少女灵体阶段后两天。

**内容：**
- 清晨。老麦赶马车去镇上。买煮酒用的粮食、食材。挑东西的时候还是不多说话——但他会多看一眼粮食的成色。
- 同一时间，酒馆里。少女 Sylvena 灵体浮在吧台边。她用魔法擦酒柜——书浮在空中，手指挥动，酒瓶一颗一颗自动排列整齐。猫形态趴在窗台上，尾巴慢慢摆。
- 老麦回来。推开门。酒柜已经干干净净。他看 Sylvena 一眼。什么也没说。把买回来的食材放在吧台上。
- Sylvena 看着他放下的食材袋子——里面多了一块小鱼干。不是不小心掉进去的。
- 窗外麦田在风里动。安静，但这次不是一个人的安静。

### 动画 3 — 所有人的打烊（全角色剧情完成后，哲学神出现前）

**触发条件：** Stage 3，所有 guest 角色剧情线完成，哲学神尚未出现。

**内容：**
- 酒馆里不止老麦一个人。所有角色以各自的方式出现在同一个场景中——
  - 警探坐在他的老位置，枪套挂在椅背上。
  - 僵尸靠着墙角，手里端着一杯——杯子里是水兑了一点酒。
  - 牛仔靠在吧台边，靴子翘在另一张凳子上。
  - 猫女（猫形态）趴在窗台上，吊坠微微发光。如果已是少女灵体以上，灵体浮在猫上方。
  - 维修师在角落摊开一张图纸，但今天没人在修东西。
  - 赌徒和死神在角落下一盘棋。赌徒在笑，死神不动。
- 老麦在吧台后面。倒酒。擦杯子。和平时一样——但这次他不是一个人。
- 有人说了什么（听不清），老麦嘴角动了一下。不是 Stage 1 的"我听到了"，是 Stage 3 的——他在笑。
- 窗外的麦田和平时一样安静。墙上的钟在走。

## Cross-character Links

- **警探 Lendro：** 对冲的两个人。Lendro 话多是为了防安静，老麦沉默是因为安静本来就没不好。Lv.1 就会感受到——一个说一车话，一个擦一只杯子。到后面 Lendro 不说话的时候，老麦会注意到——这才是变化。
- **猫女 Sylvena：** 酒保不定义她，这是她信任酒馆的原因。动画 2 中两人的无声协作，是老麦对"家人"最接近的定义。
- **牛仔：** 老麦是她的锚点之一——她跑遍宇宙，每次回来，吧台后面还是同一个人。这个"不变"在 Stage 3 中会被观众重新理解：不是冷漠不变，是选择不离开。
- **僵尸：** 老麦给他从水倒到稀释的酒——不是测试，是观察。僵尸学会叫他的名字时，老麦会应。这个"应"在 Stage 1 不会发生。
- **维修师：** 退休工人和修东西的人——有共同语言。都是动手的人，都懂机器。老麦不问技术细节，但聊起车间的事能聊两句。偶尔会多倒一点酒——"修东西的人容易渴。"
- **赌徒 Aletor：** 结局 B 碰杯时，老麦的位置和动作是场景的情感支点。
- **死神 Lethus：** 老麦会正常生老病死——他只是在这个特定时间点恰好在这间酒馆里。死神是规则：无情、高效、偶尔喝酒也是疏离的。对所有人都是规则，对老麦也不例外。
- **哲学神：** 哲学神来到酒馆——也许是他选择了酒馆，也许酒馆就是他理想中的样子。老麦和所有碎片一样，不需要对神负责。老麦的回应——倒一杯酒。推过去。不是深沉，是对谁都一样。Stage 3 的他可能会补一句："酒快凉了。"
- **机器人：** 酒馆里唯一一个和他一样不属于"任何一面墙"的存在。他被创造为创造者，老麦被需求为见证者。

## Ending Conditions

老麦没有 guest 角色的 Ending A/B。他的"结局"不是选择，是状态——

- **唯一终点：** 所有角色剧情完成后，钟还在走。夜班继续。没有大结局，没有关店。他只是承认了——这些人他在乎。
- **开放式收束：** 动画 3 就是他的"结局"。不是离别，是所有人都在。平时各讲各的故事，今晚在同一盏灯下。

如果某一天钟不走了——他不知道会发生什么。擦完最后一只杯子，挂好围裙，关掉灯，推门出去。外面是什么，他不知道，也不急着知道。

## Forbidden Canon Drift

- 不得把老麦写成"智慧老人"——他的洞察力不来自形而上的哲学，而是来自于千锤百炼的生活阅历和极强的现实逻辑。他看破，但不说破。
- 不得让他对客人说教。他不是老师，他是倒酒的。
- 不得让他主动讲述自己的故事——除非被客人直接问到，且 Stage 2 以上。
- 不得让他在 Stage 1 就表现出明显的关心。距离是他花了二十二年建起来的壳。
- 不得把他的“少言”写成“愚钝”或“冷血”。他的内心极其热诚，只是由于缺乏感性表达技巧，才选择用精准的物理关怀（如留灯、递水）代替语言。
- 不得让他的"冷笑话"变成话痨——Stage 3 他仍然话少，只是不再用沉默挡人。
- 不得把"夜班"写成神秘、诗意或哲理概念——对他来说"夜班"就是夜班。去上夜班的时候外面也是黑的、也是没人。没区别。
- 不得让他变成"团队的爸爸/导师/精神支柱"——他是见证者，不是组织者。
