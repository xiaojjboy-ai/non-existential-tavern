# 希尔维娜（Sylvena）— Character Canon

## Core Identity

一只不知何时出现在酒馆门口的紫色小猫，戴着一顶深紫色魔法帽，脖子上系着一块水晶吊坠。平时趴在窗台边，捡食客人不要的残渣。但当酒保用安尼尔的药酒配出魔法药酒后，她在半夜偷喝——紫光大作，一个身穿魔法袍的灵体以猫为锚点显现。

她不是升级变强，而是在逐步还原。猫 → 小女孩 → 少女 → 中年女性，是她的灵体从模糊到清晰的投影。最终站在你面前的那个中年女人，就是她原本的样子——一位曾经大获成功、偏执而锋芒毕露的女魔法师。

**核心命题**：永恒的存在，但是错误的永恒。

## Immutable Canon

- Sylvena 从很小就展现出惊人的魔法天赋，被学院分到最为自由但也最为怪异的"探索魔法道路"班级。她像那种不告诉你过程、直接给出答案的学霸，对正常交流兴趣寥寥，无视了所有人。
- 她在魔法典籍中发现了"灵魂魔法"，弄到了学院珍藏的灵魂宝石项链并激活了它。法阵生效时，一只猫从窗外扑向那颗闪亮的宝石——灵魂法术产生了完全不可控的质变。猫成了 Sylvena 永恒的灵体囚笼。
- 灵魂寄生在猫上后，她的魔法天赋几乎归零，但知识体系和学习能力还在。她通过逆向手段，以灵体方式把自己召唤出来，显示在猫的身体上方。
- 在她还是少女时期、天赋登峰造极时，曾随手制作了一根传送法杖——课堂练习级别的作品。这根法杖后来被赌徒 Aletor 借走用于逃离死神的追捕。赌徒选择直面死亡时，法杖被物归原主，留在了酒馆。
- 猫的灵力储存能力低下，灵体显现需要外部灵力持续供能——酒馆的魔法药酒是关键。
- 只有最后以中年女性完全体显现时，她才能第一次感受到"后悔"。但她无法命名它——她没有体会过这个情绪，不知道这个是"后悔"。

## Core Wound / False Belief / True Need

- `core_wound`：灵魂寄生在猫上——不是被敌人打败，是被自己的野心和一场意外困住。她获得了永恒，但永恒是一个错误。
- `false_belief`："天才不会质疑自己行为的正确性。"——她用傲气和偏执挡住所有质疑，包括来自自己的。
- `true_need`：学会感受那个"很重"的东西——不是战胜后悔，是承认它的存在并与之共处。或者不共处——选择永远留在窗台边，等待一颗不会来的流星。
- `core_question`：一个永恒的天才，能学会后悔吗？如果学会了，永恒会变得可以忍受，还是更痛苦？

## Lv 体系

| Lv | 形态 | 性格 | 状态 |
|----|------|------|------|
| 1-2 | **猫** | 无人类表达，仅靠本能 | 记忆无法保存，灵体完全沉睡 |
| 3-4 | **小女孩灵体** | 敏感、怯懦、语焉不详、情绪化 | 灵魂刚被唤醒，意识模糊。穿着不合身的学院袍子 |
| 5-6 | **少女灵体** | 多疑、睿智、细节了如指掌、常识浑然不知、魔法天赋登峰造极、有傲气 | 逐渐恢复记忆与能力，有学院导师证明可给出 |
| 7 | **魔女（中年女性完全体）** | 偏执、成熟、知分寸、知进退、但依旧有锋芒 | 完整记忆恢复，第一次感受到"后悔"但无法命名 |

### 立绘解锁
- Lv.1：猫形态（基础）
- Lv.3：小女孩灵体
- Lv.5：少女灵体
- Lv.7：魔女完全体

### 升级机制：打烊互动阶段

安尼尔 Stage 2 药酒交易后，每天打烊前酒馆进入互动阶段。玩家可选择对窗台的猫进行：

- **摸猫**：猫好感微增，不推动 Lv
- **逗猫**：猫好感微增，不推动 Lv
- **喂酒**：4选1酒品，其中包含**魔法药酒**（老麦用安尼尔的药酒配制）

每次喂魔法药酒 → sylvena_level +1。其他选项不推动 Lv。

**升级路径（最优，每天喂魔法药酒）**：
```
Stage 2 当天（打烊喂酒）→ Lv.1→2（猫）
次日（打烊喂酒）       → Lv.2→3（小女孩！首次人形）
第3天（打烊喂酒）      → Lv.3→4（小女孩）
第4天（打烊喂酒）      → Lv.4→5（少女！← cat_awakening_day 记录）
第5天（打烊喂酒）      → Lv.5→6（少女）
第6天（打烊喂酒）      → Lv.6→7（魔女！完全体）
第7天 = 安尼尔 Stage 3
```

## Stage List

猫女的 Stage 不按来访次数划分——她一直都在酒馆。她的 Stage 映射的是 Lv 里程碑和与其他角色（尤其是安尼尔）的交集。

| Stage | Lv 范围 | 形态 | 与安尼尔的交集 |
|-------|---------|------|--------------|
| 1 | Lv.1-2 | 猫 | Stage 1：紫猫在窗台观察。Stage 2：药酒交易后对药酒瓶产生兴趣 |
| 2 | Lv.3-4 | 小女孩 | Stage 3 分支B：简单怯生交流，问图纸是什么 |
| 3 | Lv.5-6 | 少女 | Stage 3 分支C：完整学院袍子反应链。Stage 4：给出导师印记 → 结局3「苗」 |
| 4 | Lv.7 | 魔女 | 猫女自身结局触发：客座教授 / 窗台的守望者 |

---

### Stage 1 — 窗台的猫（Lv.1-2）

紫猫趴在酒馆靠窗的桌子。戴魔法帽，挂水晶吊坠。眼神灵动，不像一只猫——更像一个正在审视酒保和整个酒馆的督察。

**老麦视角：**
安尼尔 Stage 1 首次入场前，紫猫先跳上吧台。老麦跟猫对视，看出了它眼里审视的意味。一次无声的对峙。老麦认可了这只猫：

> "这年头客人来也不定会花钱。小客人我就不招待你了，你自己找地方呆着吧。"

猫围着吧台走了一圈，跳下去，在靠窗的桌子趴下。

**安尼尔 Stage 2 后：**
药酒瓶放在吧台角落——猫够不着但能看见。猫开始对那个瓶子产生无法解释的兴趣。她不知道自己为什么盯着它。但水晶吊坠偶尔泛一下微光。

**允许：**
- 猫的日常行为：趴窗台、舔毛、看窗外、偶尔喵一声
- 对特定物品（药酒瓶、魔法相关的东西）表现出异常的专注
- 水晶吊坠在特定时刻泛光——微弱，但玩家会注意到

**禁止：**
- 不要让人形灵体在 Lv.1-2 出现
- 不要让猫做出明显超出猫范畴的智能行为
- 不要让猫直接与人"交流"

---

### Stage 2 — 小女孩的初次问候（Lv.3-4）

第一次喂魔法药酒后，晚上猫偷喝药酒昏迷，紫光大作——一个小女孩的灵体首次显现。

**小女孩的状态：**
- 敏感、怯懦、语焉不详、情绪化
- 穿着不合身的学院法师袍——太大，袖口卷了好几道
- 说话断断续续，可能只说几个词，然后就缩回猫里
- 记忆无法完整保存，每次出现可能不记得上次说过什么

**安尼尔 Stage 3 分支B 的互动：**
- 对安尼尔感到好奇但不敢靠近
- 躲在猫的身体后面，只露出半张脸
- 简单几句交流："……你是修东西的？""……那个图纸，是什么？"
- 问完就缩回去，不会展开对话

**允许：**
- 小女孩偶尔在打烊后出现，不限于安尼尔来访日
- 对酒馆里的人和物表现出孩子气的好奇
- 情绪化——可能突然笑，也可能突然沉默

**禁止：**
- 不要展现"天才"或"傲气"——那是少女阶段才有的
- 不要展开关于学院、魔法的长篇对话
- 不要让小女孩给出任何关键信息或道具

---

### Stage 3 — 少女的镜子（Lv.5-6）

持续喂魔法药酒到 Lv.5——猫女以少女灵体完全显现。学院法师袍合身，眼神锐利。多疑、睿智、细节了如指掌、常识浑然不知。魔法天赋登峰造极。

**少女的状态：**
- 还不知道自己不能恢复。保持着傲气，以及天才般的直觉
- 看着满身油污的安尼尔，大概也是不以为然的
- 对技术细节有天然的好奇——"传送阵供能？那个效率很低。你改过符文布局吗？"
- 持有学院导师证明——一枚边角有磨痕的魔法印记

**安尼尔 Stage 3 分支C：**
- 安尼尔惊诧于袍子、年纪、灵体状态 → 触发完整学院回忆链
- 天赋的镜子：她是天才，他是庸才。都穿着学院袍子，一个被追着要回去，一个被劝退赶出来

**安尼尔 Stage 4 结局3「苗」：**
- 少女靠窗坐着，一直看窗外，像在听又像没在听
- 安尼尔讲完三方现状后，她随手丢来魔法印记
- "拿去用吧。"——不多解释
- 她不知道安尼尔为什么愣住。就像她不知道自己为什么醒过来——只知道那天晚上有人留下了一瓶药酒

**允许：**
- 少女对图纸的技术好奇，可能提出一两个专业问题
- 保持傲气和距离感——她不是来交朋友的
- 多疑——对老麦的调酒、对安尼尔的动机、对酒馆的存在本身

**禁止：**
- 不要让她对安尼尔表示同情或理解（她还没学会这些）
- 不要让她主动讲述自己的过去（那是魔女阶段才做的事）
- 不要把印记的给予写成"感动"或"被说服"——她只是随手一丢

---

### Stage 4 — 魔女的抉择（Lv.7）

连续7天喂魔法药酒——猫女以中年女性完全体显现。完整记忆恢复，偏执、成熟、知分寸、知进退、依旧有锋芒。

她第一次感受到那个"很重"的东西。但她不知道那是"后悔"。

**示例对话：**

> "我恢复了所有法术知识，所有魔法理解，所有我曾经掌握的东西。但有一种……感觉，我不知道它是什么。它不像是法术失败的挫败，不像是研究卡住的烦躁。它更像……每当我看到那只猫——我的身体——趴在窗台上，看着窗外，我就……"
>
> *她停顿了很久。*
>
> "我就希望那个窗外的世界，还能让我有想要跳出去的冲动。但我没有。我只觉得……重。"

**此时触发猫女自身结局分支。**

## 猫女结局

猫女自身结局在 Lv.7（魔女完全体）时触发。与安尼尔的结局是两套独立系统——安尼尔 Stage 4 的结局3「苗」只需 Lv.5+ 少女在场即可触发，不依赖猫女自身结局走向。

### 结局A「客座教授」— 面对后悔

**条件：** sylvena_level == 7 + 玩家在关键对话中选择引导她面对

**过程：**
- 在 Lv.7 的最终对话中，她重新审视自己——从"天才的傲慢"到"接受错误"
- 玩家选择"那是后悔"——她摇头否认，但沉默后轻声说："……也许吧。"
- 她第一次真正理解那个"很重"的感觉，并学会与之共处

**最终去向：**
- 她回到魔法学院，成为灵魂魔法和动物魔法研究教授
- 不再逃避自己的"错误永恒"，而是将它转化为研究的动力
- 她曾经因灵魂魔法而毁灭，现在用知识去面对它

**与女星际牛仔的互文：**
猫女结局A与牛仔结局A「锚点与约定」互为正反馈——和解则双赢。

> 牛仔："等我飞完这趟……我会回来。"
> 猫女："……你最好记得。"

猫女看到了"走出去"的可能性——有人在漂泊中能找到锚点，她也能回去面对学院。两人约定一定要回来跟对方坐坐。

**主题**：和解。一个否定自己的天才，最终学会了——错误本身也是永恒的一部分。

---

### 结局B「窗台的守望者」— 留在酒馆

**条件：** sylvena_level == 7 + 玩家在关键对话中未引导她面对（或她选择逃避）

**过程：**
- 她不想面对任何东西——学院、过去、那个"很重"的感觉
- 她赖在了酒馆，成为了常驻角色
- 她偶尔以猫的形态趴在窗台，偶尔以魔女的灵体形态出现

**与女星际牛仔的互文：**
猫女结局B与牛仔结局B「虚无与流星」互为负反馈——决裂则双输。大纷争中两人互相否定对方的存在方式，不欢而散。牛仔开着"流星"离开酒馆，再也不会回来。

**最终画面：**

> 她坐在酒馆的窗台边，透过那扇窗——时间与空间之外的窗——看到的是一片虚无。
>
> 她坐在虚无前，等待。
>
> 她在想：虚无里是否会有一颗流星划过？

**"流星"的三重含义：**
1. 她童年许愿时看到的流星
2. 牛仔飞船的名字"流星"——那个约定了回来坐坐、但再也不会来的人
3. 一个再也不会来的承诺

**主题**：否定。否定他人就是否定自己。一个永恒的天才，最终困在了自己建造的第二个囚笼里。第一个囚笼是猫，第二个囚笼是酒馆。而这一次，是她自己选择的。当你在虚无中等待一颗流星时，你等待的不是归来，而是自己不敢面对的东西。

---

### 猫女结局与安尼尔结局的关系

猫女的结局与安尼尔结局3「苗」**互不阻塞**：

- 安尼尔结局3「苗」只需 **sylvena_level >= 5**（少女），在安尼尔 Stage 4 触发
- 猫女自身结局需 **sylvena_level == 7**（魔女），在猫女 Stage 4 触发
- 两者可以同时达成：少女在安尼尔 Stage 4 给出印记（苗结局），之后猫女继续升级到 Lv.7，触发自身结局
- 也可以只达其一：玩家喂到 Lv.5-6 后停止 → 只有苗结局，猫女自身结局不触发

### 双结局对照

| 维度 | 结局A「客座教授」 | 结局B「窗台的守望者」 |
|------|-------------------|----------------------|
| 面对/逃避 | 面对自己的后悔 | 逃避那个"很重"的感觉 |
| 永恒的意义 | 将错误转化为知识 | 在否定中消解意义 |
| 窗外是什么 | 回到学院（重新进入世界） | 看到虚无（彻底抽离） |
| 与酒保的关系 | 感谢与告别 | 依赖但沉默 |
| 最后的形态 | 完全的中年女魔法师（真身） | 猫与灵体交替的不稳定状态 |
| 核心情感 | 和解 | 等待 |
| 与牛仔的关系 | 和解——约定回来坐坐 | 决裂——牛仔再也不会回来 |

## 标志性道具

- **魔法帽**：深紫色，从猫形态就戴着。是她学院时期的物品，缩小后系在猫头上。
- **水晶吊坠**：灵魂宝石的碎片，挂在猫脖子上。是她灵体的锚点。在灵力波动时会泛微光。
- **学院导师证明（魔法印记）**：Lv.5+ 少女持有。边角有磨痕，像是被反复摸过。在安尼尔结局3「苗」中丢给他——"拿去用吧。"

## Cross-character Links

- **安尼尔（维修师）：** 核心关系。两个"魔法体系"的人，天赋完全相反——她是天才，他是庸才。Stage 2 他留下的药酒是她苏醒的起点。Stage 4 她丢出的印记是他第三条路的钥匙。两人都不知道对方为什么帮自己——药酒因果回环。
- **老麦（酒保）：** 他用安尼尔的药酒配出魔法药酒，放在吧台角落。猫偷喝后显现灵体。他是唯一一个不试图定义她的人，这正是她逐渐信任的原因。
- **赌徒 Aletor：** 她少女时期随手做的传送法杖被他借走用于逃离死神。赌徒用"活着"的囚笼关住存在的意义，猫女用"酒馆"的囚笼关住面对自己的勇气——两个囚笼，两个永恒的错误。赌徒选择直面死亡时，法杖被物归原主。
- **死神 Lethus（薛定谔的猫）：** 在死神的视角中，Sylvena 处于死和未死的叠加态。猫的躯体存活且被魔法灵力持续修复，已化为灵猫；猫女的灵体永恒存在。在死神的清单上，她的名字既不能被"划掉"，也不能被"确认"——她存在于死亡的观测之外。（此概念不直接在游戏对话中点出，交由玩家自行领悟。）
- **女星际牛仔（镜像关系）：** 两人的命运完全挂钩——和解则双赢，决裂则双输。猫女的"被困"和牛仔的"流浪"是同一枚硬币的两面：猫女被永恒困在猫的躯体里想逃出去，牛仔被扔在无限宇宙里找不到能停下来的地方。两人都渴望对方拥有的东西，但都不愿意承认。牛仔的4次来访与猫女的Lv阶段严格挂钩：猫形态（初遇投喂）、小女孩灵体（掏枪震惊）、少女灵体（互相看不顺眼但看见共同点）、魔女（大纷争决定结局走向）。结局A「客座教授」↔ 牛仔结局A「锚点与约定」（约定回来坐坐）。结局B「窗台的守望者」↔ 牛仔结局B「虚无与流星」（决裂，流星再也不会来）。"流星"是牛仔飞船的名字，也是猫女童年许愿时看到的流星——一个再也不会来的承诺。
- **创造性思维机器人：** 一个是被变成猫的天才，一个是被创造出来的AI——"被限制的形态"的共鸣。

## Ending Conditions

猫女的结局系统独立于安尼尔的结局系统，但存在交叉点：

**安尼尔结局3「苗」：**
- 条件：安尼尔 Stage 4 + sylvena_level >= 5（少女）
- 触发：猫女少女随手丢出学院导师证明给安尼尔
- 不依赖猫女自身结局走向

**猫女自身结局：**
- 条件：sylvena_level == 7（魔女完全体）
- 结局A「客座教授」：关键对话中选择引导她面对后悔
- 结局B「窗台的守望者」：关键对话中未引导面对，或她选择逃避

**两者关系：**
- 互不阻塞，可以同时达成
- 玩家可能只触发其中一个（取决于喂食天数和对话选择）

## Forbidden Canon Drift

- 不得把猫女写成"被酒保治好的傲娇"。酒保不治疗任何人。
- 不得让猫形态（Lv.1-2）展现出人类智能——她是猫，不是披着猫皮的人。
- 不得让小女孩（Lv.3-4）展现出天才或傲气——那是少女阶段才有的。
- 不得让少女（Lv.5-6）对安尼尔表示同情或理解——她还没学会这些情感。
- 不得把"后悔"的情感提前到少女阶段——后悔只有在魔女完全体（Lv.7）时才能第一次感受到，且她无法命名它。
- 不得在结局A写成"皆大欢喜"——她学会了后悔，但永恒的错误依然存在。
- 不得在结局B写成"悲惨结局"——她选择了窗台，这是她自己的选择。等不到流星也是一种活法。
- 不得让印记的给出（结局3「苗」）带有"感动"或"被说服"的情绪——她只是随手一丢。"拿去用吧。"
