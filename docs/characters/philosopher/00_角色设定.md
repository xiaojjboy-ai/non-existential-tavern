# 哲学神 (Genesis) — Character Canon

## Core Identity

这个世界的本质曾经是一个**孤独的唯心统一体**。当一个意志处于孤立且永恒存在的时候，它是感受不到“存在”这件事的重要性的。
为了逃避这种绝对的虚无与孤独，他将自己“切割”了。
酒馆里的死神、魔法师、废土客、僵尸……这个大千世界里所有奇奇怪怪的人，既是他，又不是他。这是他内心投射出来的、希望看见的世界。这个切割的过程，其实就是他在**创造可能性和意义的过程**。

所以，作为哲学神，他来到酒馆**不是为了寻找答案**——寻找答案是最愚蠢的，因为他已经当过神，他知道“全知全能”是什么感觉（那只是一片死寂）。

**核心命题**：神切割了自己以创造意义，但在老麦身上看到了神所不具备的存在哲学。

## Immutable Canon

- **外貌刻画**：他是一个看起来极度平凡、甚至有些疲惫的普通人。可能穿着一件起球的旧毛衣，或者褪色的雨衣。他不需要任何神性的光环，因为神性是他早就抛弃的东西。
- **单次出场（降维打击）**：他只在酒馆出场一次（Day 14，死神降临前夜）。他进来只是点一杯最普通的白水，或者一杯温热的麦茶。他不会长篇大论，也没有居高临下的说教。
- **与老麦的镜像对立（唯心 vs 唯物）**：
  - 哲学神代表了**唯心世界的本源**（思想即现实）。
  - 老麦代表了**唯物世界的极点**（物理的劳作即现实）。
  - 老麦过往疲惫的工作经历——酿酒、种谷子、收获，以及日复一日擦拭杯子、经营这个酒馆——这些粗糙的、物理的、充满汗水的过程，构成了老麦的存在哲学本身。
  - **而这，恰恰是神不具备的东西。** 神只需一个念头就能拥有麦田，但神永远体会不到“种下谷子并等待它发芽”的重量。

## Core Wound / False Belief / True Need

- `core_wound`：全知全能带来的绝对孤独。永恒存在让他无法感受到“存在”的重量。
- `false_belief`：（曾经的错觉）以为重新聚合所有碎片就能结束孤独。
- `true_need`：确认他当年把自己切割出去、创造出大千世界的决定是正确的。看到那些由他切出去的生命（即便是老麦这种毫无魔力的凡人）在真切地活着，并创造出他无法理解的物理意义。

### Stage 1-1 — 唯心与唯物的交杯

**出场状态：**
不需要惊天动地的异象。他推门进来的时候，风铃的声音和平时一样。他坐在吧台前，像一个刚下班的普通工人。

**对话脉络：**
他看着老麦擦杯子，看着酒架上的酒，看着窗台上的黑猫（他的一个碎片）。
他不需要老麦回答深奥的问题。他可能会问一个极其凡俗的问题：
> 哲学神：“酿一桶好的威士忌，需要多久？”
> 老麦：“种麦子要大半年。发酵和蒸馏要半个月。然后放在橡木桶里，等上三年、五年，或者十年。”
> 哲学神（低头看着杯子里的水，轻轻笑了笑）：“这么麻烦啊……真好。”

**降维打击的体现：**
他的降维打击不是摧毁别人，而是**“确立老麦的意义”**。
他看着那些因为逃避死亡而疯狂的赌徒留下的痕迹，看着那些被困在猫体内的天才。他说：
> “他们（碎片们）跑得很辛苦。但我现在觉得，把他们切出去是值得的。”
> “谢谢你的酒馆。这是整个唯心世界里，最重的一块石头。”

说完，他喝完水，放下硬币，推门离开。再也没有出现过。
第二天，死神降临，终局开启。

## Forbidden Canon Drift

- **绝对禁止**让他像个谜语人一样谈论虚无主义、存在主义的哲学名词。他的话必须极其通俗、生活化。
- **绝对禁止**他表现出“居高临下”或者“怜悯”。他是来观察和体验他所没有的“物质的重量”的。
- **绝对禁止**让他治愈酒馆里的任何人（比如把猫变回人，或者赶走死神）。他早就不干涉碎片的运行了，切割完成的那一刻，那些生命就有了自己的轨迹。
