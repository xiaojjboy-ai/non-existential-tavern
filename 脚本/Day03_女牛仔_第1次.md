## 对话层

## 第一幕：雇佣兵的降临

```
[dlg_03_01]
（伴随着一声沉重的推门声，冷凝的空气灌进了酒馆。）
Cowgirl：（大步走进来。动作极度高效，没有多余的观察。她的视线只在一瞬间扫过了酒馆的几个死角，然后直接定在了吧台上。）
Cowgirl：（她穿着皮夹克，身姿挺拔，但透着一种随遇而安的洒脱。像是星际航道上习惯了随时和死神擦肩而过的雇佣兵。）
Cowgirl：（走到吧台前，随手将一个沾满星际尘埃的金属终端拍在桌面上。）
Cowgirl：这地方连个停泊引导信号都没有，还得靠肉眼找灯光。
```

### 选项 ch_03_01：老麦面对她的入场

```
[dlg_03_02a]
老麦：（手里擦着杯子，头都没抬。）
老麦：看来你的飞船没赶上禁酒令。
Cowgirl：（愣了一下，随后发出一声极其干脆的短笑。）
Cowgirl：禁酒令？那玩意在猎户座旋臂外连擦鞋都不够格。

[dlg_03_02b]
老麦：（把一块干净的垫酒巾推到她面前。动作沉稳，不带任何敌意，也不带热情。）
老麦：这儿只有酒。没有悬赏令。
Cowgirl：（看了一眼垫酒巾，眼神里闪过一丝锐利的赞赏，紧绷的肩膀微微放松了一点。）
Cowgirl：最好是。我今天不想拔枪。

[dlg_03_02c]
老麦：（把手里擦好的酒杯放下。）
老麦：太空里也有这种假酒？
Cowgirl：（挑了挑眉毛，拉开吧台椅坐下，动作利落。）
Cowgirl：假酒喝不死人。真空中那些看起来很美的东西才会。
```

## 落座与烈酒需求

```
[dlg_03_03]
Cowgirl：（她靠在椅背上，那是种随时能弹射起步、又仿佛要在椅子上睡死过去的矛盾坐姿。）
Cowgirl：给我来杯烈的。越烈越好。
Cowgirl：最好能一口把嗓子烧穿，让我忘了这见鬼的导航系统故障。
```

## 调酒反馈

```
[dlg_03_drink_correct]
（老麦转身，倒了一杯纯白酒。没有冰块，没有点缀。推过去。）
Cowgirl：（端起杯子，一饮而尽。连眉头都没皱一下。）
Cowgirl：够劲。比第三星区那些掺了工业废料的引擎燃料强多了。
（她把空杯子顿在桌上。这一口酒下去，她身上那种在宇宙里漂流太久的沧桑感稍微被压了下去。）

[dlg_03_drink_wrong_whisky]
（老麦倒了一杯威士忌。推过去。）
Cowgirl：（闻了一下，喝了一大口。）
Cowgirl：橡木桶的味道。太文雅了，不够野。
Cowgirl：不过在这个连信号都没有的鬼地方，也算不错了。

[dlg_03_drink_wrong_beer]
（老麦倒了一杯啤酒。推过去。）
Cowgirl：（看着上面浮着的泡沫，冷笑了一声。）
Cowgirl：啤酒？你想让我喝一肚子水，然后半夜在休眠舱里憋醒吗？
（她还是喝了半杯，但显然没有被满足。）

[dlg_03_drink_wrong_sweet]
（老麦倒了一杯颜色鲜艳的果酒。推过去。）
Cowgirl：（死死盯着那杯酒，手指在腰间的枪套上敲了两下。）
Cowgirl：你当我是来参加学院派的毕业舞会吗，老头？
```

## 第二幕：猫的突袭与本能

```
[dlg_03_04]
Cowgirl：（她正准备要第二杯，余光突然扫到了吧台最远端的阴影。）
Cowgirl：哟。这儿还有只活物。
（那是老麦养的猫。此刻正闭着眼睛，尾巴不耐烦地甩动了一下。）
Cowgirl：（她伸出手，带着一种女人特有的随意和逗弄心，想去摸猫的头顶。）
（突然！黑猫猛地睁眼，前爪像一道黑色的闪电般划过空气，直奔她的手背。）
（但就在那一瞬间——）
Cowgirl：（她的手以一种极其违背常理的速度缩了回来。不是害怕的躲闪，而是身体本能在察觉危险前就已经做出的战术规避。）
（她看着自己手背上仅有的一道极细的红痕，没有发怒，也没有尴尬。）
Cowgirl：（她干脆地笑了一声，带着一种对自己的冷嘲。）
Cowgirl：神经反应比半人马座的掠夺者还快。
Cowgirl：（她收回手，不再去碰那只猫，而是拍了拍吧台。）
Cowgirl：脾气挺大。在太空里，脾气大的活物通常死得最早。但也有些能活成传奇。
老麦：（继续擦着第二个杯子。）
老麦：它只喝水。
```

## 第三幕：飞船、流星与虚无

```
[dlg_03_05]
Cowgirl：（她从口袋里掏出一个打火机把玩着，打火机表面磨损严重，上面刻着一颗变形的流星。）
Cowgirl：老头子，你这店开在什么见鬼的坐标上？
Cowgirl：我的飞船，那艘叫“流星”的老破船，雷达已经一礼拜没扫到过大于一立方的活人聚居地了。
Cowgirl：这感觉就像……（她停顿了一下，眼睛看着空无一物的打火机火苗。）
Cowgirl：就像整个宇宙里，就剩我这一坨会呼吸的废铁。
（她不谈什么孤独，也不谈哲学。她只谈飞船和雷达。但那种茫然的沧桑感，像冷空气一样从吧台对面渗过来。）
Cowgirl：每天醒来，检查维生系统，核对航向。做完这些，只剩安静。
Cowgirl：你知道最怕什么吗？不是陨石。不是星盗。
Cowgirl：是安静。
```

### 选项 ch_03_02：老麦的回应

```
[dlg_03_06a]
老麦：（他拿起刚才被她喝空的酒杯，一言不发地重新倒满白酒，稳稳地推到她面前。）
老麦：酒馆不负责修雷达。但能对付安静。
Cowgirl：（看着满上的酒，她眼中的茫然退去了几分，重新换上了那副干练的雇佣兵面具。）
Cowgirl：这句广告词不错。

[dlg_03_06b]
老麦：（他手里拿着抹布，把她刚才放终端时蹭出的灰尘仔细擦掉。）
老麦：这地方比你的飞船还破。但我还在擦桌子。
Cowgirl：（她看着老麦那种踏实到极点的动作，突然嗤笑了一声。）
Cowgirl：是啊。既然还没死，活儿就得继续干。

[dlg_03_06c]
老麦：（递酒过去的时候，随口跟了一句。）
老麦：放心。这杯里没加星尘。
Cowgirl：（接过酒杯，那种宏大的虚无感被这句生活化的玩笑瞬间消解。）
Cowgirl：算你有良心。星尘那玩意儿卡嗓子。
```

## 退场

```
[dlg_03_07]
（Cowgirl 喝完了第二杯酒。她没有像普通酒客那样感叹或者逗留。）
Cowgirl：（站起身，把打火机揣回口袋，动作毫不拖泥带水。）
Cowgirl：我得回去修那破雷达了。
Cowgirl：（她走向门口，皮靴踩在地板上，发出沉实的声音。在门前，她稍微停了一下。）
Cowgirl：我活着就是一切。
（这句话极轻。不是对老麦说的，更像是看着外面无尽的黑暗，给自己的一句口令。）
Cowgirl：灯亮着。我会再来的。

[narr_03_01]
伴随着厚重的关门声，那种真空中特有的冷凝气味逐渐消散。
猫舔了舔爪子，重新趴回了阴影里。
老麦收走了那个带着干涸酒渍的杯子。
杯底有一道轻微的划痕。那是雇佣兵的手指在极度用力时留下的。
```
