## 对话层

## 入场

```
[dlg_01_01]
Lendro：（推门进来，风衣肩头湿了。扫了一眼酒馆——先看屋里，然后窗，然后门。职业习惯。）
Lendro：还开着？好。这天气——关门的酒吧比开着的多。
```

### 选项 ch_01_01：老麦怎么接

```
[dlg_01_02a]
老麦：（抬头。确认"有人进来"那个级别的看。下巴朝吧台方向动了一下，不到一厘米。）
老麦：嗯。
（继续擦杯子。）
Lendro：（不需要客套——舒服了。他最怕的就是太热情。走向吧台。）

[dlg_01_02b]
老麦：（擦杯子。没抬头。但杯子往吧台中央推了一下——给酒腾了个位置。）
Lendro：（注意到这个动作。嘴角动了一下。走向吧台。）

[dlg_01_02c]
老麦：（抬头。看了一眼他的肩膀——湿了半截。）
老麦：淋得不轻。
Lendro：（低头看了看自己的肩膀。没想到会被注意到。）
Lendro：……习惯了。街上就我一个。
（走向吧台。）
```

## 落座

```
[dlg_01_03]
Lendro：（拉凳子坐下。帽子摘下来，放在右手边第二个凳子。手指在吧台上敲了三下——哒、哒、哒。不是紧张，是警察的习惯：到新地方，先碰一下最稳的东西。）
Lendro：有什么能喝的。
（不是问句。是陈述。像在说"给我一个理由多坐几分钟"。）
```

## 调酒 drink_01_01 反应

```
[dlg_01_drink_correct]
（老麦倒白酒。一口杯。满的。推过去。不说话。）
Lendro：（端起来，看了看杯中液体的颜色）
Lendro：……清的。
（小口抿了一下。）
Lendro：还行。
（肩线放下来半寸。不是好喝——是够了。）

[dlg_01_drink_wrong_whisky]
（老麦倒威士忌。推过去。看了他一眼——他知道这不是最好的选择，但不纠正。）
Lendro：（端起来闻了一下）
Lendro：……太重了。
（还是喝了一口。放下时杯底磕得比正常响。）
Lendro：有清一点的吗。

[dlg_01_drink_wrong_beer]
（老麦倒啤酒。推过去。顿了一下——这个人不像喝啤酒的，但他不替人决定。）
Lendro：（看了一眼，没端）
Lendro：……算了。
Lendro：有烈的吗。

[dlg_01_drink_wrong_lemon]
（老麦倒柠檬水。推过去。手指在杯沿停了一下——是这个人的选择，和他判断的不一样。）
Lendro：（端起来喝了一口。表情没变。）
Lendro：……你这儿就这些？
（放下杯子。手指在吧台上无声地敲了两下。）
```

## 开启话题 — Lendro问起酒馆

```
[dlg_01_04]
Lendro：（白酒喝到半杯。暖和过来了。开始说话——不是倾诉，是让对话别落地。Stage 1 独角戏模式：他不等回应，自己能撑满。）
Lendro：这地方以前没见过。新开的？
```

### 选项 ch_01_02

```
[dlg_01_05a]
老麦：三年。
Lendro：三年？我在这片跑了二十年，没路过过。怪了。
（老麦没解释。继续擦杯子。）
Lendro：（看他没展开。不追问。换方向。）
Lendro：你这吧台的木头比我办公桌好。局里的桌子——四条腿有三条垫着书。

[dlg_01_05b]
老麦：（擦杯子。停了一下。）
老麦：算新的。也不算。
Lendro：怎么说？
老麦：这地方开开关关的。换过人。上一个是个邮差。
Lendro：邮差开酒吧？
老麦：嗯。
（Lendro等了两秒。老麦在擦杯子。不展开了。）
Lendro：……行。那我换个问题——你以前干什么的。不是邮差吧。
老麦：汽车零件。冲压车间。二十二年。
Lendro：（看他的手——不是吧台磨的。是机器磨的。）
Lendro：看出来了。

[dlg_01_05c]
（老麦擦杯子。没说话。但也没避开视线。）
Lendro：（自己接上了——不需要答案的聊天。）
Lendro：你不说我也知道。开在没人路过的地方，还三年——你不是靠客人赚钱的。
（喝了一口酒。）
Lendro：你是自己想开。
（老麦看了他一眼。没承认。没否认。擦杯子。）
Lendro：没事。我不问。我今晚就是——路过。
```

## 转折 — 老麦第一次主动开口

```
[dlg_01_06]
（安静了两秒。雨声。Lendro 正准备发下一个球——）
老麦：你是警察。
Lendro：（愣了一下。笑了——不是客套的笑，是真的觉得好玩。但眼睛还是不笑。）
Lendro：怎么看出来的。
老麦：帽子放右手边。伸手能够到。不能挡视线。
（擦杯子。）
老麦：还有你进来的时候，先扫了一圈屋里。然后是窗。然后是门。
Lendro：（看着老麦。停了半秒。）
Lendro：……你自己以前也是警察？
老麦：冲压车间。机器不会跑。但人会出错。
Lendro：出事的多吗。
老麦：多。手指。胳膊。有时候更糟。
（擦杯子。节奏没变。）
老麦：所以看人。谁今天手抖了。谁昨天没睡好。线长不看这个。我看。
Lendro：（端起白酒。没喝。看着老麦。）
Lendro：你在这擦杯子，跟踩踏板差不多。
老麦：（嘴角动了一下。最轻的笑。）
老麦：没那么响。
```

## 核心对话 — Lendro 讲一个搞笑的案子

```
[dlg_01_07]
Lendro：（又喝了一口。酒过半杯。气氛对了——不是"被倾听"那种对，是"旁边这个人和我一样见过破事"那种对。）
Lendro：你刚才说看人——跟你说个事。
Lendro：上周逮了个私酒贩子。
（老麦没接话。但擦杯子的节奏慢了一点——在听。）
Lendro：把酒藏在教堂的洗礼池里。我进去的时候，他站在池子边上，跟我说——他是来受洗的。
Lendro：我说你衣服怎么是干的。他说还没开始。
（自己先笑了。讲笑话——眼睛不笑。他在填空间。）
Lendro：我说行。那这池子里是什么。他说是圣水。我说圣水怎么是棕色的。他说——新配方。
（摇头。）
Lendro：我问他叫什么。他说"我受了洗"。我说这是名字？他说——对。我受了洗，然后卖威士忌。上帝不会怪我的。
Lendro：写报告的时候我手在抖。笑的。
```

### 选项 ch_01_03：老麦怎么回应

```
[dlg_01_08a]
老麦：（嘴角动了一下。最轻的笑。）
老麦：你们这行比我那行麻烦。
Lendro：（得到了确认——他在听，他觉得好笑。deflection -5。）
Lendro：你那边呢？车间里总有什么吧。
老麦：有个哥们。每天带一样的饭盒。一样的三明治。一样的苹果。吃了十四年。
Lendro：十四年？
老麦：嗯。后来发现他不是爱吃。是他老婆只做这一种。
（擦杯子。）
老麦：他没说。十四年没说。
Lendro：（顿了一下。没笑。笑不出来了——不是不好笑，是听懂了。）
Lendro：……行。十四年的三明治。
（端起杯子。没喝，但手指放松了。）

[dlg_01_08b]
（老麦擦杯子。没笑。但杯子往Lendro那边推了半寸。）
Lendro：（注意到了。不需要他说出来。）
Lendro：你知道吗，抓他的时候他在唱圣歌。真的。被铐上还在唱。
Lendro：我说你唱什么。他说——赞美上帝。上帝让我多卖了几瓶。
（摇头。笑出声。但眼睛——还是没跟上。）
Lendro：这人要是当警察，比我强。心理素质。

[dlg_01_08c]
（老麦停下擦杯子。拿起自己的杯子——他给自己倒了一杯。举了一下。没碰，只是举了一下。）
Lendro：（看着老麦自己喝了一口。他认识这个动作——不是敬酒，是"我听到了"。）
Lendro：……行。你也听够了。不讲了。
（把杯子里的酒喝完。比平时慢。）
```

## 收尾

```
[dlg_01_09]
Lendro：（喝完最后一口。看杯底——确认喝完了。Stage 1 的仪式：翻到最后一页，看一眼，合上。）
Lendro：该走了。
（顿了一下。差点说"还不错"。改了口。）
Lendro：这地方——
（看了老麦一眼。老麦在擦杯子。没催。没问。）
Lendro：还开着吧。
```

### 选项 ch_01_04：告别

```
[dlg_01_farewell_a]
老麦：（点头。擦杯子。）
老麦：下雨就开着。
Lendro：（帽子压低。）
Lendro：……行。
（推门出去。没回头。）

[dlg_01_farewell_b]
老麦：（看了他一眼。）
老麦：嗯。你随时来。
Lendro：（顿了一下。手指在门框上敲了一下——和吧台上敲三下同一个节奏。）
Lendro：……下次下雨。
（推门出去。）

[dlg_01_farewell_c]
（老麦点头。把空杯往自己这边挪了半寸——收走了。）
Lendro：（看了这个动作。没说什么。）
（推门出去。）
```
