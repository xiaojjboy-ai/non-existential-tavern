## 对话层

## 第一幕：以“盘查”为名的试探

```
[dlg_01_01]
Lendro：（推门进来。没摘帽子。手插在风衣口袋里，视线极其锐利地扫过整个酒馆的结构。）
Lendro：我刚才跟街角的搭档说，我看到了一家连个药房招牌都不挂、直接大摇大摆亮着灯的违禁酒馆。
Lendro：（走到吧台前，手指敲了两下桌面。）
Lendro：别紧张，例行检查。所以，这是南区黑帮的场子，还是卡彭的暗桩？
```

### 选项 ch_01_01：老麦的反应

```
[dlg_01_02a]
老麦：（抬头看了一眼他的风衣。没接黑帮的话茬，只拿出一块干净的垫酒巾放在吧台上。）
老麦：客人。
Lendro：（眯起眼睛盯了老麦两秒，突然咧了下嘴，但不带笑意。）
Lendro：反侦察意识不错。行。

[dlg_01_02b]
老麦：（没停下手里的活，继续擦杯子。）
老麦：这儿只有酒。没有暗桩。
Lendro：（手指在吧台上重重敲了一下。）
Lendro：这年头，卖酒就是最大的暗桩。

[dlg_01_02c]
老麦：（停下擦杯子，直视着他。）
老麦：如果我说都不是，你信么。
Lendro：（冷笑一声，拉开高脚凳坐下。）
Lendro：我只信物理证据。
```

## 落座与“没收物证”

```
[dlg_01_03]
Lendro：（把帽子摘下，放在右手边的吧台上。）
Lendro：既然你说这儿只卖酒，那给我倒一杯你们这儿最烈的。
Lendro：（他双手交叉，目光紧盯着老麦的手。）
Lendro：作为“证物”。我倒要看看你这卖的是什么工业酒精。
```

## 调酒反馈

```
[dlg_01_drink_correct]
（老麦转身，倒了一杯纯正的白酒。一口杯，没有冰，没有任何花哨。推过去。）
Lendro：（端起来闻了一下，紧绷的肩膀往下放了半寸。）
Lendro：……没兑水。也没兑劣质糖浆。
（他一口喝掉半杯。这是他今晚第一口不是为了“工作”而喝的东西。）

[dlg_01_drink_wrong_whisky]
（老麦倒了一杯威士忌。推过去。）
Lendro：（端起来闻了一下。）
Lendro：威士忌？在黑店喝这个可是赌命。
（但他还是喝了一大口。眉头皱了一下。）
Lendro：重了点。但起码没下毒。

[dlg_01_drink_wrong_beer]
（老麦倒了一杯啤酒。推过去。）
Lendro：（看了一眼，完全没碰。）
Lendro：我在查违禁烈酒，你给我一杯工人喝的啤酒？
Lendro：（叹了口气，敲敲吧台）
Lendro：来点清的、烈的。别像个磨洋工的修女。

[dlg_01_drink_wrong_lemon]
（老麦倒了一杯柠檬水。推过去。）
Lendro：（死死盯着那杯水，像是在看一个极其恶劣的笑话。）
Lendro：……你是在羞辱我，还是在羞辱禁酒令？
```

## 第二幕：看破身份与工业区侧写

```
[dlg_01_04]
Lendro：（拿着杯子转了一圈，目光在老麦身上上下打量，突然用一种确定的语气开口。）
Lendro：你的虎口没有老茧，指腹和手掌却全是粗糙的死皮。那不是拿枪的手。
老麦：（手上的动作没停，只是静静地擦着酒杯边缘。）
Lendro：你站的姿势重心在两腿中间，而不是偏向惯用手——你不带枪，也不打算随时拔枪。
Lendro：如果外面有警察砸门，你的第一反应绝对是去护着这些酒瓶，而不是去摸吧台底下的喷子。
老麦：（放下擦好的杯子，平淡地看着他。）
Lendro：所以，你不是帮派分子。你就是个真正的……该死的酒保。
Lendro：（他凑近了一点，盯着老麦的手和肩膀，开始喋喋不休的评判。）
Lendro：再看看你这糟糕的形象和松垮的样子……像极了工业区里那种几个月没休息过的伙计。
老麦：（嘴角不置可否地动了一下。）
Lendro：不，不是“像极了”。简直就是。
Lendro：应该说，你就是他们几十年以后的样子。被机器榨干了，跑到死胡同里开个见不得光的酒馆养老。
```

### 选项 ch_01_02：老麦回应工业区的侧写

```
[dlg_01_05a]
老麦：（放下擦杯子的布。从吧台底下摸出一盒烟，抽出一根递过去。）
老麦：冲压车间。干了二十二年。
Lendro：（没有犹豫，接过烟咬在嘴里。老麦给他点上。这是警探对安全环境的认可。）
Lendro：二十二年。难怪你身上有股铁锈味。

[dlg_01_05b]
老麦：（点了点头，动作很慢，像是关节里真的生了锈。）
老麦：以前是在工厂里。机器停了，人就只能找别的事干。
Lendro：（哼了一声，掏出打火机给自己点了一根烟。）
Lendro：机器停了，你还能开个店。有的人机器停了，人也就跟着停了。

[dlg_01_05c]
老麦：（没承认，也没否认，只是微微颔首。拿过一块干净的抹布继续擦拭木头吧台。）
Lendro：（看着老麦沉稳的动作，警惕感彻底解除。）
Lendro：不反驳就是承认。你们这种老骨头我见得多了。
```

## 第三幕：窗外黑影与虚无主义漫谈

```
[dlg_01_06]
Lendro：（伸手进大衣内袋，摸出一张钞票，啪地拍在吧台上。）
Lendro：再来一杯。这次我付钱。
老麦：（收走钞票，转身去拿酒瓶。）
（突然，窗外一道极快的动物黑影飞掠而过。无声无息。）
Lendro：（夹着烟的手在半空中顿了一下。他盯着窗外空无一物的夜色，慢慢吐出一口烟圈。）
Lendro：那是什么东西……跑得真快。连只流浪猫都像在被债主追。
老麦：（把第二杯酒推到他面前，依旧没说话。）
Lendro：（端起酒杯，语速开始加快，带着浓浓的抱怨。）
Lendro：这该死的地方总是有些不速之客。你知道吗？今天局里的接线员非得让我去处理一个报警。
Lendro：有个老太太发誓说，她邻居家的狗每天半夜在唱爵士乐。我去了一看——那条狗还真的在嚎叫。音准差得要命。
老麦：（拿起一块抹布，擦拭着根本没有灰尘的吧台。）
Lendro：我就站在那儿听了一条狗嚎了十分钟。不仅是我，我的搭档也站在那儿。我们就这么浪费了生命中宝贵的十分钟。
Lendro：这种荒谬的日常，和我们在黑夜里窜来窜去的流浪猫有什么区别？自以为有方向，其实都在瞎折腾。
老麦：（擦拭的动作微微放慢。）
Lendro：你知道我在看那些抢臭鱼的帮派分子时在想什么吗？
Lendro：人类意识绝对是宇宙进化史上的一个恶劣玩笑。我们就像长在发霉面包上的绿毛，明明毫无意义，却偏要每天早上爬起来，去给另一撮发霉的绿毛开罚单。
Lendro：宇宙是一片死寂的深渊。我们在深渊里折腾，就像刚才那只猫，以为跳过了窗台，其实还是在烂泥里。
```

## 第四幕：邪教徒闯入与隐秘交锋

```
[dlg_01_07]
（就在 Lendro 念叨“深渊”时，酒馆的门被猛地撞开了。）
（一个神情极度紧张、眼眶深陷的邪教徒冲了进来。他满脸狂热，径直冲向 Lendro。）
邪教徒：（声音发抖）你看到了！你看到了真实的深渊！您……您就是接引的高阶祭司吗？！
Lendro：（端着杯子，像看智障一样看着他。但他握杯子的手指明显用力了。）
邪教徒：（激动得手舞足蹈）主在深渊中呼唤！那些不可名状的伟大苏醒了！星辰就要归位了！
Lendro：（脸色微变，死死盯着手里的杯子。几秒后，他猛地转过头，拔高了声音。）
Lendro：深渊？码头底下发臭的排水沟也叫深渊？别拿这种糊弄三岁小孩的鬼话来浪费我的时间！
Lendro：什么星辰？你是说南区那个叫“星辰”的走私帮派？
Lendro：听着，小子。我不管你嘴里的“主”是哪个高利贷头子，你现在最大的麻烦是你嗑药嗑出了幻觉。对吧？
```

### 选项 ch_01_03：老麦如何处理这场荒诞

```
[dlg_01_08a]
老麦：（拿起一块抹布，平静地擦着吧台，完全无视了邪教徒的狂热。只对 Lendro 说。）
老麦：你的烟灰要掉吧台上了。
Lendro：（配合着老麦，故意把烟灰弹进烟灰缸里，吸引了邪教徒的注意力。）
Lendro：听见没？酒保嫌你吵了，小子。

[dlg_01_08b]
老麦：（看了一眼邪教徒，从吧台下倒了一杯冰水，重重地墩在邪教徒面前。）
老麦：喝了。闭嘴。
邪教徒：（被老麦的气场震慑，微微瑟缩了一下，但依然死死盯着 Lendro。）
Lendro：（冷眼看着邪教徒。）
Lendro：喝点水吧，你的嘴唇干得像砂纸，我都怕你突然碎在地板上。

[dlg_01_08c]
老麦：（什么也没说，只是静静地退后了一步，把这个舞台完全留给了 Lendro。）
Lendro：（看老麦不管事，只能自己叹了口气，转向邪教徒。）
Lendro：行吧。既然酒保不想理你，那我理理你。
```

## 毒辣侧写与暗中交接

```
[dlg_01_09]
邪教徒：（他警惕地看了一眼四周，突然压低身体，凑到 Lendro 身边。）
邪教徒：（压低声音，快速且隐秘地将一个沾满暗绿色黏液的雕像，硬生生滑进了 Lendro 的大衣口袋里。）
邪教徒：伟大的祭司……信物给您。仪式就靠您了！深渊万岁！
Lendro：（顺势用大衣挡住邪教徒的手，突然大声呵斥。）
Lendro：别把你的脏手往我身上蹭！滚蛋！
（邪教徒以为完成了使命，像见鬼一样转头冲出酒馆，门被摔得震天响。）
（酒馆里重新恢复了死寂。）
Lendro：（他端着酒杯，沉默了足足五秒。然后，他慢慢放下酒杯，用两根手指，从大衣口袋里夹出了那个黏糊糊的旧神信物。）
Lendro：看看现在的年轻混混，连发疯都发得这么毫无创意。满嘴全是三流地摊文学里的词。
Lendro：脚步虚浮，肌肉痉挛，瞳孔散大得能塞进一枚硬币，起码三天没合过眼了。更要命的是，他鞋底那种带油污的红泥，只有码头东区才有。
Lendro：（他盯着老麦，用刑警的语气一锤定音。）
Lendro：他偷了人家极其重要的东西，现在，这东西在我手里了。
（就在信物暴露在空气中的那一刻，酒馆里的温度似乎骤降了几度。一种无形的、令人作呕的深海腥味弥漫开来。）
Lendro：（用力揉了揉太阳穴，死死咬着牙对老麦说。）
Lendro：该死，这鬼东西带着一股让人头晕的恶臭。味道真冲。
```

### 选项 ch_01_04：诡异氛围（SanCheck）与老麦的回应

```
[dlg_01_farewell_a]
老麦：（老麦的动作停顿了一下。他感受到了空气中那种违背物理常识的压迫感。但他没有抬头。）
老麦：空气里有死鱼的味道。很危险。
Lendro：（看着老麦这副如临大敌却依然坚守吧台的样子，嘴角勾起一抹危险的笑意。）
Lendro：危险？我最喜欢危险的味道。
Lendro：（他用一块手帕包起信物，塞进口袋。拉了拉大衣的领子。）
Lendro：没想到我只是想钓几条小鱼解解馋，结果直接上了条大鱼。
Lendro：（走到门边，回头看了一眼老麦。）
Lendro：老骨头，明天最好还亮着灯。我还会来。

[dlg_01_farewell_b]
老麦：（眉头微皱，不动声色地把一盘用来驱散异味的干柠檬片推到了 Lendro 面前。）
老麦：这东西不属于街头。把它扔了。
Lendro：（他低头看了一眼干柠檬片，眼神柔和了半秒，但很快又被冷酷的职业本能覆盖。）
Lendro：扔了？这可是能让东区翻天覆地的钥匙。
Lendro：（他把信物仔细包好。大步走向门口。）
Lendro：本来我只是想翘班喝杯酒，钓几条小鱼。现在看来，今晚这网捞了条大鲨鱼啊。
Lendro：（推开门。）
Lendro：酒不错。明天我还会来。希望那时候你还没关门。

[dlg_01_farewell_c]
老麦：（在这股令人作呕的空气中，老麦一言不发，默默地给 Lendro 的空杯子里倒了一个杯底的烈酒。这是为了压惊。）
Lendro：（感受到空气中那种几乎让人发疯的低频震动，他没有犹豫，端起那杯底的酒，一饮而尽。辛辣的酒精强行拉回了他的理智。）
Lendro：谢了。够劲。
Lendro：（他把信物收好，仿佛那只是个普通的赃物。）
Lendro：我这趟翘班真值。本想钓个小偷小摸的小鱼解馋，竟然有这种大鱼自己撞上网来。
Lendro：（拉开门。）
Lendro：明天我还会来。别急着打烊，酒保。
```
