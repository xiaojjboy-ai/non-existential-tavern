## 对话层

## 入场

```
[dlg_01_01]
Lendro：（推门进来，风衣肩头湿了。没立刻往里走，而是站在门口。视线像扫地雷一样扫过酒馆——先看天花板角落，然后窗，然后门。职业习惯。）
Lendro：还开着？好。这天气——关门的酒吧比开着的多。
```

### 选项 ch_01_01：老麦怎么接

```
[dlg_01_02a]
老麦：（抬头。确认"有人进来"那个级别的看。下巴朝吧台方向动了一下，不到一厘米。）
老麦：嗯。
（继续擦杯子。）
Lendro：（不需要客套。舒服了。他最怕的就是太过热情的酒保。走向吧台。）

[dlg_01_02b]
老麦：（擦杯子。没抬头。但杯子往吧台中央推了一下——给酒腾了个位置。）
Lendro：（注意到了这个动作。嘴角动了一下。走向吧台。）

[dlg_01_02c]
老麦：（抬头。看了一眼他的肩膀——湿了半截。）
老麦：淋得不轻。
Lendro：（低头看了看自己的肩膀。似乎没想到会被注意到。）
Lendro：……习惯了。街上就我一个像傻子一样在雨里走。
（走向吧台。）
```

## 落座与要求

```
[dlg_01_03]
Lendro：（拉凳子坐下。帽子摘下来，放在右手边第二个凳子。手指在吧台上敲了三下——哒、哒、哒。不是紧张，是某种神经质的习惯动作，确认这地方稳不稳当。）
Lendro：有什么能喝的？不要那些五颜六色带甜味的，随便来点什么。
（他语速很快，像在赶时间，又像只是受不了没人说话。）
```

## 调酒反馈

```
[dlg_01_drink_correct]
（老麦倒白酒。一口杯。满的。推过去。没说话。）
Lendro：（端起来，看了看杯中液体的颜色）
Lendro：……清的。
（小口抿了一下。）
Lendro：还行。不辣嗓子。
（紧绷的肩膀往下放了半寸。第一杯酒总是最重要的锚点。）

[dlg_01_drink_wrong_whisky]
（老麦倒威士忌。推过去。他知道这不是最好的选择，但不替客人做决定。）
Lendro：（端起来闻了一下）
Lendro：……太重了。
（但他还是喝了一口。放下时杯底磕得比正常响。）
Lendro：下次还是换点清的吧。我怕喝完不知道自己叫什么。

[dlg_01_drink_wrong_beer]
（老麦倒啤酒。推过去。这通常是体力劳动者的选择。）
Lendro：（看了一眼，没端）
Lendro：……算了。这玩意喝多了肚子里全是气。
Lendro：有烈一点、清一点的吗？

[dlg_01_drink_wrong_lemon]
（老麦倒柠檬水。推过去。手指在杯沿停了一下——是这个人的选择，尽管不合理。）
Lendro：（端起来喝了一口。表情没变，但眉毛挑得老高。）
Lendro：……你这儿就这些？
（放下杯子。手指在吧台上无声地敲了两下。像是在怀疑自己走错了地方。）
```

## 开启话题 — 神经质的碎嘴子

```
[dlg_01_04]
Lendro：（暖和过来了一点。开始说话——语速不减，不是为了倾诉，只是单纯地在填满空间。）
Lendro：这地方以前没见过。新开的？你这招牌连个灯都不亮，也就是我这种下雨天迷路的人能撞进来。
```

### 选项 ch_01_02

```
[dlg_01_05a]
老麦：三年。
Lendro：三年？我在这片跑了二十年，没路过过。真是见鬼了。
（老麦没解释。继续擦杯子。）
Lendro：（看老麦是个闷葫芦，不以为意，自顾自换了个方向。）
Lendro：不过你这吧台的木头倒是比我办公桌好。局里的桌子——四条腿有三条全靠垫旧报纸。

[dlg_01_05b]
老麦：（擦杯子。停了一下。）
老麦：上一个老板是个邮差。
Lendro：邮差开酒吧？哈！难怪信总是送错地方。
老麦：嗯。
（Lendro等了两秒，看老麦不往下说，自己就接了过去。）
Lendro：……行。我不问邮差去哪了。那你以前干什么的？看着不像干服务业的。
老麦：汽车零件。冲压车间。二十二年。
Lendro：（看他的手——不是吧台磨的。是极其粗糙的老茧。）
Lendro：看出来了。

[dlg_01_05c]
（老麦擦杯子。没说话。但也没避开视线。就这么听着他说。）
Lendro：（完全不需要答案，自己接上了。）
Lendro：开在没人路过的死胡同，还三年——你肯定不是靠卖酒赚钱的。
（喝了一口。）
Lendro：你是自己想待在这儿。
（老麦看了他一眼。没承认，没否认。）
Lendro：没事。我不问。我今晚也就是个——路人。
```

## 转折 — 老麦的观察力

```
[dlg_01_06]
（安静了两秒。外面雨声淅沥。Lendro 正准备开启下一个莫名其妙的话题——）
老麦：你是警察。
Lendro：（愣了一下。然后咧嘴笑了——不是客套，是真的觉得有意思，尽管眼睛没笑。）
Lendro：怎么看出来的？我把警徽戴头上了？
老麦：帽子放右手边第二个凳子。伸手刚好能够到，又不会挡视线。
（擦杯子。）
老麦：还有你进来的时候，看屋子的顺序。先看窗，再看门，最后看人。
Lendro：（死死盯着老麦。停顿了足足三秒。这种反常的安静在他身上很少见。）
Lendro：……你自己以前也是警察？
老麦：冲压车间。每天看人。看谁今天手抖了，看谁昨天没睡好。
（节奏没变。）
老麦：所以看人准。
Lendro：（端起酒杯，又放回去。他其实有点心虚，因为这意味着他的掩饰被看穿了。）
Lendro：你在这擦杯子，可比在车间里安全多了。
```

## 核心对话 — 教堂洗礼池的走私案（黑色幽默）

```
[dlg_01_07]
Lendro：（又喝了一口酒。气氛松了点。他需要证明自己真的只是个普通的、每天抓小偷的警察。）
Lendro：你刚才说看人准——跟你说个真事。
Lendro：上周，我们队在码头区逮了个私酒贩子。
（老麦没接话，但擦杯子的动作放慢了——表示在听。这是游戏核心机制：倾听比追问更有力量。）
Lendro：那孙子把酒藏在破教堂的洗礼池里。我带人踹门进去的时候，他正站在水池边上，一本正经地跟我说——警官，我是来受洗的。
Lendro：我说你他妈衣服怎么是干的？他说还没开始。
（Lendro 自己先笑了出来。笑得很夸张，像在酒馆里表演单口相声。）
Lendro：我说行。那这池子里是什么？他说是圣水。我说圣水怎么是棕色的还冒泡？他说——这是新配方，上帝昨晚托梦告诉他的。
（摇头，语气荒诞。）
Lendro：我就问他叫什么名字。他说“我受了洗”。我说这是名字？他说——对。我受了洗，然后卖威士忌。上帝不会怪我的。因为他保佑了我的生意。
Lendro：写报告的时候我手都在抖。我不知道我是气抖的，还是笑抖的。
```

### 选项 ch_01_03：老麦怎么回应

```
[dlg_01_08a]
老麦：（嘴角动了一下。最轻的笑。没有任何同情，纯粹是被荒诞逗笑。）
老麦：你们这行，比我那行麻烦得多。
Lendro：（得到了极好的确认——他在听，而且他没深究。警探觉得很安全。）
Lendro：麻烦？哈！麻烦那是往轻了说。
老麦：车间里最麻烦的，就是每天带一样的饭盒，吃一样的苹果，吃了十四年的人。最后发现他根本不爱吃苹果。只是没人听他说。
Lendro：（本来想接烂话，突然顿了一下。没笑出来。他听懂了隐喻。）
Lendro：……行。十四年的苹果。
（端起杯子。沉默地喝掉了一口。）

[dlg_01_08b]
（老麦擦杯子。没笑。但杯子往 Lendro 那边推了半寸。）
Lendro：（注意到了这个极其微小但肯定的动作。他觉得这酒保简直是个绝佳的树洞。）
Lendro：你知道最扯的是什么吗？抓他上车的时候，他还在唱圣歌。真的。被铐上还在唱。
Lendro：我说你唱什么呢。他说——赞美上帝，上帝让我多卖了三瓶。
（摇头。长叹了一口气，带点疲惫。）
Lendro：这人要是当警察，心理素质绝对比我强。

[dlg_01_08c]
（老麦停下擦杯子。拿起自己的杯子——给自己倒了一杯水。举了一下。没碰杯，只是举了一下。）
Lendro：（看着老麦自己喝了一口。他认识这个动作——这是“我在听你吹牛”的最高礼遇。）
Lendro：……行了。你也听够了这种狗屁倒灶的破事了。
（把杯子里的酒喝完。刚才的神经质消散了不少。）
```

## 收尾

```
[dlg_01_09]
Lendro：（喝完最后一口。看杯底——确认喝完了。把杯子重重放在木吧台上。）
Lendro：酒不错。比码头那些掺水的假货强。
（他站起来，抓起帽子。动作干脆利落。他不习惯逗留。）
Lendro：这地方——
（看了老麦一眼。老麦在安静地等他说完。）
Lendro：还开着吧？我是说，明天。
```

### 选项 ch_01_04：告别

```
[dlg_01_farewell_a]
老麦：（点头。拿着毛巾擦吧台。）
老麦：下雨就开着。
Lendro：（把帽子压低，遮住一半眼睛。）
Lendro：……行。那这鬼天气最好别停。
（推门出去。没回头。）

[dlg_01_farewell_b]
老麦：（看了他一眼。）
老麦：随时来。
Lendro：（顿了一下。手指在门框上无意识地敲了一下——和他刚坐下时敲吧台的节奏一模一样。）
Lendro：……等下次下雨吧。
（推门出去。）

[dlg_01_farewell_c]
（老麦点头。把空杯往自己这边挪了半寸——收走了。一句话没说。）
Lendro：（看了这个动作。这是最舒服的距离感。）
Lendro：走了。
（推门出去。）
```

## 退场旁白与猫咪伏笔

```
[narr_01_01]
吧台上留着一圈水渍——分不清是雨水还是融化的冰。
老麦拿起空杯。擦干。看了一眼墙上的钟。
10:43。还早。
就在他转身去整理酒架时，听到窗玻璃传来一声极轻的“沙啦”刮擦声。
老麦回头。窗外只有倾盆的暴雨和无边的黑夜。
什么都没有。
但当他凑近时，在窗玻璃的最下角，看到了一枚小小的、带点泥污的猫爪印。
印在玻璃外侧。
但在这种连成年人都几乎站不稳的狂风暴雨里，怎么可能会有猫静静地蹲在窗台上？
```
