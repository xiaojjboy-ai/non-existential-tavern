## 对话层

### 入场

```
[dlg_01_01]
Lendro：……还开着？
```

#### 选项 ch_01_01 回应

```
[dlg_01_02a]
酒保：开着。坐吧。

[dlg_01_02b]
（酒保擦杯子，不说话）

[dlg_01_02c]
酒保：你淋湿了。
```

### 落座

```
[dlg_01_03]
Lendro：（拉高脚凳，坐下，动作像在审讯室里找位置）
Lendro：嗯。
（风衣扣子解开，帽子摘下放在膝上）
Lendro：有什么能喝的。
```

#### 选项 ch_01_02 回应

```
[dlg_01_04a]
Lendro：随便来点什么。

[dlg_01_04b]
Lendro：随便来点什么。

[dlg_01_04c]
Lendro：（顿了一下，低头看了看自己的肩膀）
Lendro：……习惯了。
（拉凳子坐下，比前两个选择慢半拍）
Lendro：有什么暖的。
```

### 调酒 drink_01_01 反应

```
[dlg_01_drink_correct]
Lendro：（端起来，没喝，看了看杯中液体的颜色）
Lendro：……清的。
（小口抿了一下）
Lendro：还行。
（微微放松，肩线下降约一指宽）

[dlg_01_drink_wrong_whisky]
Lendro：（端起来闻了一下）
Lendro：……太重了。
（还是喝了一口，放下时杯底磕得比正常响）
Lendro：有清一点的吗。

[dlg_01_drink_wrong_beer]
Lendro：（看了一眼，没端）
Lendro：……算了。
Lendro：有烈的吗。

[dlg_01_drink_wrong_lemon]
Lendro：（端起来喝了一口，表情没变）
Lendro：……你这儿就这些？
（放下杯子，手指在吧台上无声地敲了两下）
```

### 核心对话 ① — 码头

```
[dlg_01_05]
Lendro：（白酒喝到半杯，声音比进门时低了一档）
Lendro：这地方……以前没见过。
```

#### 选项 ch_01_03 回应

```
[dlg_01_06a]
酒保：新开的。

[dlg_01_06b]
酒保：你以前常去哪？
Lendro：（手指摩挲杯沿）
Lendro：码头那边。第三街拐角有个地下室。
（停顿）
Lendro：……不去了。
（视线短暂移向窗外北墙方向，随即收回）

[dlg_01_06c]
（酒保继续擦杯子）
```

```
[dlg_01_07]
Lendro：我是个警察。或者说——曾经是。
（喝一口酒）
Lendro：二十年了。码头区，禁酒令，走私……都是我的辖区。
Lendro：你见过码头凌晨三点的样子吗？
```

#### 选项 ch_01_04 回应

```
[dlg_01_08a]
酒保：没见过。

[dlg_01_08b]
酒保：三点怎么了？

[dlg_01_08c]
（安静等他说）
```

```
（无论选什么，Lendro 都接以下台词。停顿长度不同：A正常 / B更紧 / C最长）

Lendro：雾。从水面升起来，把整个码头吞进去。
Lendro：仓库的灯一盏一盏灭掉，像是有什么东西在吹蜡烛。
（手指无意识地摸了一下风衣内侧——枪套的位置）
Lendro：……那天也是这样的雾。
```

### 核心对话 ② — 仓库轮廓

```
[dlg_01_warehouse]
Lendro：（酒已经见底，但没有要续的意思）
Lendro：有个搭档。叫 Eilas。
（停顿很长，像是在决定要不要说出这个名字）
Lendro：比我年轻五岁。刚从学院出来的时候，是我带的他。
（握杯的手指关节发白）
Lendro：码头第十七号仓库。线人说那批货当晚到。
Lendro：我们两个人进去的。
（停顿两秒）
Lendro：……出来的只有我一个。
```

#### 选项 ch_01_05 回应

```
[dlg_01_eilas_a]
酒保：发生了什么？
Lendro：（看了你一眼，像是在判断你是不是审讯室里的人）
Lendro：……你真想听？
（不等回答）
Lendro：仓库里不是走私酒。是——
（突然停住，喉结动了一下）
Lendro：……我不记得了。
（声音变硬）
Lendro：不。我记得。我只是——
（停顿）
Lendro：……还没准备好说。
（防御姿态，肩线重新绷紧）

[dlg_01_eilas_b]
酒保：你不用说了。
Lendro：（明显松了一口气，但嘴角没有上扬）
Lendro：……谢了。
（把空杯推过来，手指在杯口停了一瞬）
Lendro：下次吧。
（退回安全距离）

[dlg_01_eilas_c]
（酒保给他倒酒，不说话）
Lendro：（看着新倒的酒，没立刻端起来）
Lendro：……
Lendro：（低声）你倒酒的时候不问问题。
（端起来，小口喝）
Lendro：……这样挺好。
（放松程度最高）
```

### 收尾

```
[dlg_01_exit]
Lendro：（不管之前选了什么，最终都会起身）
Lendro：……该走了。
（风衣扣子扣上，戴上帽子，帽檐压得很低）
Lendro：这地方——
（停顿）
Lendro：……还开着吧？
```

#### 选项 ch_01_06 告别

```
[dlg_01_farewell_a]
酒保：一直开着。
Lendro：（点头，推门离开）

[dlg_01_farewell_b]
酒保：你随时来。
Lendro：（顿了一下，没回头，推门离开）

[dlg_01_farewell_c]
（酒保点头）
Lendro：（推门离开）
```
