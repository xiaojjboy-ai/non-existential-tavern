# 指令层模板 — 怎么写控制流

## 这层是干什么的？

指令层就是"导演"。它负责告诉游戏：

- 现在画面是什么
- 音乐放什么
- 谁走进来了
- 什么时候停顿
- 玩家选了A/B/C之后跳到哪段对话

**指令层里没有台词，没有动作描写，只有"命令"。**

---

## 所有标签一览

| 标签 | 干什么 | 怎么写 | 例子 |
|------|--------|--------|------|
| `[BG]` | 切背景图 | `[BG 资源名]` | `[BG tavern_night_rain]` |
| `[BGM]` | 放背景音乐 | `[BGM 资源名]` | `[BGM jazz_piano_low]` |
| `[BGM fade_out]` | 音乐渐弱消失 | `[BGM fade_out 秒数]` | `[BGM fade_out 3s]` |
| `[SE]` | 播一个音效 | `[SE 资源名]` | `[SE door_open_wind_chime]` |
| `[LIGHT]` | 灯光/色调描述 | `[LIGHT 自由文本]` | `[LIGHT 暖黄偏暗，角落阴影重]` |
| `[PROP]` | 场景里有什么道具 | `[PROP 自由文本]` | `[PROP 吧台干净，酒架半满]` |
| `[ENTER]` | 角色立绘出场 | `[ENTER 角色_状态]` | `[ENTER Lendro_coat_wet]` |
| `[EXIT]` | 角色立绘消失 | `[EXIT 角色]` | `[EXIT Lendro]` |
| `[EMO]` | 角色表情微变 | `[EMO 角色 描述]` | `[EMO Lendro 肩线下降]` |
| `[PAUSE]` | 停顿几秒 | `[PAUSE 秒数]` | `[PAUSE 1.5s]` |
| `[GOTO]` | 跳到某段对话 | `[GOTO 锚点ID]` | `[GOTO dlg_01_01]` |
| `[CHOICE]` | 玩家选项组 | `[CHOICE 锚点ID]` + 下面列选项 | 见下文 |
| `[END]` | 这夜脚本结束 | `[END]` | `[END]` |

---

## 选项怎么写？

选项是玩家能点的东西。格式：

```
[CHOICE ch_01_01]
  A → dlg_01_02a
  B → dlg_01_02b
  C → dlg_01_02c
```

意思：
- `ch_01_01` = 这个选项组的名字（锚点ID）
- `A → dlg_01_02a` = 玩家选A，就跳到对话层里叫 `dlg_01_02a` 的那段
- 选项数量可以是2、3、4，不超过4

**选项的文字内容不在这里写——写在对话层里。这里只写"选了之后去哪"。**

---

## 调酒怎么写？

调酒也是一种选项，只是选项内容是酒名：

```
[CHOICE drink_01_01]
  白酒   → dlg_01_drink_correct
  威士忌 → dlg_01_drink_wrong_whisky
  啤酒   → dlg_01_drink_wrong_beer
  柠檬水 → dlg_01_drink_wrong_lemon
```

意思：玩家选了白酒，跳到"正确反应"那段；选了别的，跳到对应的"错误反应"那段。

## 锚点ID怎么命名？

锚点就是"名字"，让指令层和对话层能互相找到。规则：

```
格式：{类型}_{天数两位}_{序号两位}

类型：
  dlg    → 对话节点
  ch     → 选项组
  drink  → 调酒环节
  narr   → 旁白

例子：
  dlg_01_01       → 第1天，第1段对话
  ch_01_03        → 第1天，第3个选项组
  drink_02_01     → 第2天，第1次调酒
  narr_01_01      → 第1天，第1段旁白
```

同一个选项组里的不同分支，加小写字母后缀：
- `dlg_01_02a` = 选A之后的对话
- `dlg_01_02b` = 选B之后的对话
- `dlg_01_02c` = 选C之后的对话

## 一夜的指令层长什么样？

按时间顺序写，从开场到退场：

```
## 场景初始化          → 先把画面、音乐、灯光摆好
## 入场               → 角色走进来
## 选项 ch_XX_01      → 玩家第一次选择
## 落座/开场           → 角色坐下，开始说话
## 选项 ch_XX_02      → 玩家第二次选择
## 调酒 drink_XX_01   → 给角色倒酒（如有的话）
## 核心对话 ①         → 角色讲第一层故事
## 选项 ch_XX_03      → 玩家第三次选择
## 核心对话 ②         → 角色讲第二层故事（如有）
## 选项 ch_XX_04      → 玩家第四次选择（如有）
## 收尾               → 角色起身，告别
## 选项 ch_XX_05      → 告别时的选择
## 退场               → 角色离开，旁白，结束
```

## 写的时候注意

1. **不要写台词**——台词属于对话层
2. **不要写数值**——好感度变化属于数据层
3. **每个 `[GOTO]` 指向的锚点，必须在对话层里存在**
4. **每个 `[CHOICE]` 的锚点，必须在数据层 branches 里存在**
5. 没有调酒环节的夜，整段删掉就行
6. 双人场景用两次 `[ENTER]`，写两个角色
