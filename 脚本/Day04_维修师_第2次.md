## 对话层

## 入场与铺垫

```
[dlg_04_01]
（风铃声响。安尼尔推门走进来。这次他没有在门口停顿，也没有审视酒馆。）
（他径直走向吧台，拉开上次坐过的凳子。沉重地坐下。长长地叹了一口气——像是一台老旧的锅炉在排压。）
老麦：（什么也没问，直接倒了一杯满杯的啤酒，推过去。）
安尼尔：（端起杯子，一口气喝了半杯。喉结滚动。砰地一声放下酒杯。）
安尼尔：……再来一杯。
（老麦拿过空杯，转身打酒。安尼尔从怀里掏出那张边缘磨损的图纸，熟练地在吧台上摊开。不需要再试探是否会被看到——这是一种默认的安全。）
```

## 吐槽之夜

```
[dlg_04_02]
（安尼尔盯着图纸，但手里的铅笔没有动。机油味比上次更重了。）
安尼尔：今天上头派了几个“新贵族”的监理来车间。
（他笑了一下。不是开心的笑，是那种见多了之后的荒谬感。）
安尼尔：穿着白衬衫，打着领结。站在熔炉前面三米远，捏着鼻子问我，这台机器的排气阀能不能改成镶金边的。说这样显得有“工业美学”。
（老麦把第二杯啤酒推过去。没说话。）
安尼尔：政府拨款，学院验收，新贵族分包。他们根本不在乎这台列车能不能跑。他们只在乎单子上盖了几个章。
安尼尔：我们这些在一线拧螺丝的，就像是被浪潮冲上岸的螃蟹。水退了，他们在岸上挑挑拣拣，我们只能在沙子里吐泡泡。
（他不愤怒。他只是极其疲惫。那是一种已经看透了游戏规则、却不得不继续陪玩的麻木。）
```

## 选项 ch_04_01：老麦的陪伴方式

```
[dlg_04_03a]
（老麦拿过一块干净的杯布，开始擦架子上的空杯子。没有任何语言回应。）
（安尼尔看了他一眼。老麦的安静就像一面不追问的墙。把所有的抱怨吸收，不反弹任何建议。）
安尼尔：……是啊。吐泡泡。
（他端起酒杯，又喝了一大口。紧绷的肩膀放松了一点。）

[dlg_04_03b]
老麦：（擦着吧台上的水渍。）
老麦：至少你上了岸。没被淹死。
（安尼尔停下了酒杯。转头看着老麦。然后，他低声笑了起来。笑声里带点粗糙的回音。）
安尼尔：哈……你说得对。没被淹死。我还活着。
（他拿起铅笔，在图纸上用力划了一道。）
安尼尔：只要还活着，图纸就还得画。

[dlg_04_03c]
（老麦没有说话，而是拿起酒桶的管子，把安尼尔刚才喝掉的一口啤酒又给补满了。泡沫再次溢出杯口。）
（安尼尔看着那杯重新变满的啤酒。动作比任何语言都管用——他在告诉他，不管岸上怎么样，这里的酒是满的。）
安尼尔：……谢了。
（他把手伸进西装口袋，摸索着什么。）
```

## 药酒伏笔

```
[dlg_04_04]
安尼尔：（看着杯底的泡沫）
安尼尔：其实……如果当年我不倒卖那些魔力恢复药剂，我现在可能还在学院的地下室里扫地。
（他没注意自己顺嘴说漏了什么。“学院”、“倒卖药剂”。但他不在乎了。）
安尼尔：那些号称研究真理的大法师，买起黑市的提神药水来，比车间的工人还要疯狂。
老麦：（擦杯子的手停了一下。他抓住了那两个字。）
老麦：……药剂？
```

## 强制事件：药酒交易

```
[dlg_04_05]
安尼尔：（愣了一下，似乎意识到自己说多了。但他看了一眼老麦，又看了一眼吧台。）
安尼尔：……是。以前弄过一点。怎么？
老麦：你那些药剂里——有没有药酒？
安尼尔：（皱了皱眉）
安尼尔：有。一种粗制滥造的魔力恢复草药泡的。以前在学院里没人要，后来我拿去给车间的工人。腰伤了擦一点，十分钟就能站起来。我包里刚好有一瓶车间没用完的。
（他从那个磨损的皮包里掏出一个玻璃扁瓶。深褐色的液体，但在吧台的灯光下，隐隐透着一丝暴躁的紫色微光。）
老麦：（盯着那个瓶子。）
老麦：拿来换啤酒。
安尼尔：什么？
老麦：你在我这儿的啤酒——用药酒换。
（安尼尔看着老麦。他在心里算了一笔账——这一瓶廉价药酒的成本，甚至抵不上一杯黑啤的利润。这是一笔绝对亏本的买卖。对老麦来说。）
（但他端起老麦那杯啤酒，喝了一口。因为老麦是一面不追问的墙。他不需要解释。）
安尼尔：……成交。
（他把药酒瓶推过吧台。）
```

## 收尾与退场

```
[dlg_04_06]
（交易达成。安尼尔把剩下的啤酒喝完，打了个酒嗝。机油味和酒精味混在一起。）
安尼尔：行了。明天还要看那些贵族在锅炉前假笑。我得回去了。
（他把图纸重新卷好。这次动作快了一点。扣上西服的扣子。）
老麦：慢走。
安尼尔：……下次见。
```

## 猫与药酒（伏笔）

```
[narr_04_01]
老麦拿起那瓶药酒。粗糙的玻璃瓶，标签都被机油蹭模糊了。他把它放在了吧台最靠窗的角落。
窗台上，原本正在打盹的紫猫突然睁开了眼睛。
它没有看老麦，也没有看门外。那双眼睛死死地盯着那个瓶子。脖子上的水晶吊坠，在黑暗中泛起了一阵和药酒一模一样的、深紫色的幽光。
```
