## 对话层

## 入场

```
[dlg_03_01]
（风铃声响。Lendro 推门进来。他这次没有扫视四周，而是径直走向吧台。熟门熟路。）
Lendro：这雨下得像是要把整个码头淹了。
（他拍了拍风衣上的水珠。）
```

### 选项 ch_03_01：老麦的迎接方式

```
[dlg_03_02a]
老麦：（什么也没说，只是拿出一个干净的一口杯，放在吧台上。）
Lendro：（拉开凳子坐下，把帽子放在手边。他喜欢这种不废话的默契。）

[dlg_03_02b]
老麦：还是一样？
Lendro：（拉开凳子坐下，把帽子放在手边。）
Lendro：一样。不用冰。外面够冷的了。

[dlg_03_02c]
老麦：（抬头看了一眼。）
老麦：今天没那么湿。
Lendro：（低头看了一眼自己的肩膀。今天确实打了伞。）
Lendro：跑得比上次快点。

## 调酒反馈

[dlg_03_drink_correct]
（老麦倒满白酒，推过去。）
Lendro：（看着那杯刚好倒满的清透液体。嘴角扯了一下。）
Lendro：记性不错。省了我说话的力气。

[dlg_03_drink_wrong_whisky]
（老麦倒了一杯威士忌，推过去。）
Lendro：（看了一眼琥珀色的酒液，眉头微微一皱。）
Lendro：我上次说过，这玩意太重了。
（但他还是端起来喝了一口。）

[dlg_03_drink_wrong_other]
（老麦倒了一杯不太符合警探气质的饮品，推过去。）
Lendro：（盯着杯子看了两秒。）
Lendro：……你们这种开在偏僻地方的酒吧，幽默感总是很特别。
（他无奈地叹了口气，端起来抿了一口。）
```

## 落座与寒暄

```
[dlg_03_03]
（Lendro 端起杯子，喝了一大口。熟悉的辛辣感让他紧绷的肩膀往下沉了沉。）
Lendro：本来不想绕远路的。但主街那边的酒吧，放的音乐太吵。
（他不等老麦接话，无缝接上了下一个话题。他来这里就是为了说话的。）
Lendro：说到吵——前天晚上，主街上有家钟表店被砸了。
```

## 核心案子：砸钟的失眠者

```
[dlg_03_04]
Lendro：我去的时候，那家伙拿着个扳手，站在满地玻璃碴里。店里七十二个挂钟，全让他给砸了。
（他用手指在吧台上画了个圈，像是比划满地狼藉。）
Lendro：我把他铐起来。问他图什么？偷金子还是偷机芯？
Lendro：你猜他怎么说？他说是声音太吵了。
（Lendro 突然笑了一声。笑声干巴巴的，停留在喉咙里。）
Lendro：他说，“警官，你懂不懂？时间一直在走，滴答滴答的。我想让它安静一会儿。”
（他又喝了一口酒，摇了摇头。）
Lendro：疯子。他要是不想听滴答声，去买个日晷不就行了。非得拿扳手。
```

### 选项 ch_03_02：老麦的回应

```
[dlg_03_05a]
老麦：（一边擦杯子一边开口。）
老麦：日晷晚上看不见。
Lendro：（愣了一下。然后大笑出声。这次笑得稍微真诚了一点。）
Lendro：哈！有道理。我下次审他的时候把这句写进笔录里。

[dlg_03_05b]
老麦：他砸完以后，安静了吗？
Lendro：（笑容收敛了一点。手指在杯沿上摩挲了一下。）
Lendro：没。他说他脑子里还有一堆钟在响。
（Lendro 撇了撇嘴。）
Lendro：所以我说他是疯子。

[dlg_03_05c]
（老麦没说话，只是把一个干净的杯子放回架子上，发出轻微的磕碰声。）
Lendro：（看着老麦的动作。他习惯了这种没有评价的聆听。）
Lendro：总之，又是个被这该死的城市逼疯的倒霉蛋。
```

## 环境突变：雨停与本能应激

```
[dlg_03_06]
（Lendro 讲完了案子，准备开启下一个话题。）
Lendro：对了，报纸上说最近——
（毫无征兆地，外面的暴雨突然停了。）
（酒馆背景里持续了整晚的巨大白噪音瞬间消失。空间突然陷入了极度空旷的死寂。）
（窗台上原本在打盹的紫猫猛地睁开眼睛，瞳孔竖成了一条线，背部的毛瞬间炸起。动物对危险和杀气的直觉总是比人更敏锐。）
（Lendro 的话音猛地掐断了。）
（他不是在回忆，也没有发抖。他的身体在不到半秒的时间里做出了最极致的战术应激：背部肌肉瞬间绷紧，眼神变得像鹰一样锐利而警觉。）
（他的右手以极快的速度本能地下落，“啪”地一声死死扣住了西装下的枪套柄。散发出一股随时准备拔枪的杀气。）
（在这种失去听觉掩护的极度安静里，他的潜意识判定：有危险。有人潜伏。环境不再安全。）
（他下意识地抗拒这种绝对的安静，因为这种安静和当年那个没有声音的码头仓库一模一样。）
```

### 选项 ch_03_03：老麦的物理打断

```
[dlg_03_07a]
老麦：（眼神一沉。他捕捉到了Lendro肌肉绷紧的危险信号，也余光看到了窗台炸毛的紫猫。）
（为了保护客人，也为了避免走火，老麦没有任何犹豫。他直接抓起吧台上那瓶厚重的白酒瓶，“砰”地一声重重砸在木桌面上。）
老麦：酒不够了。我得去地窖拿一瓶。
（沉闷的撞击声粗暴地撕裂了真空般的死寂。紫猫被这声巨响吓了一跳，发出一声短促的“嘶”，轻盈地跳下窗台，钻进了阴影里。）
（这声巨响和猫的动静，把 Lendro 从高度戒备的战术状态里强行拽了回来。）

[dlg_03_07b]
老麦：（他看到了那只扣在枪套上的手，也注意到了紫猫喉咙里已经开始发出极具攻击性的低吼。）
（老麦顺手拿起一个厚底的啤酒杯，倒了半杯冰水，贴着吧台表面用力一推。）
（玻璃杯摩擦着木头，发出长长的一声“唰——”，准确地滑到 Lendro 手边，轻轻撞了他的手肘一下。）
老麦：喝口水。你的酒气有点重。
（紫猫的视线被那个滑动的玻璃杯吸引，偏着头盯着水里的冰块。随着猫的注意力被转移，房间里那种一触即发的紧绷感也随之消散。冰水的温度同时打断了 Lendro 的应激循环。）

[dlg_03_07c]
老麦：（不动声色地跨前一步，挡在 Lendro 和窗台的紫猫之间，形成一个保护性的站位。）
（他顺手打开了吧台下方的制冰机盖子，抓起一把金属冰夹，故意在不锈钢桶边缘用力刮了两下。刺耳的金属摩擦声响起。）
老麦：这机器老是卡住。
（尖锐的物理噪音刺破了沉默。紫猫“喵”了一声，像是因为受不了这种刺耳的声音，不满地甩了甩尾巴，跳下了窗台。Lendro 眼里的那股狠厉也随之瞬间散了。）
```

## 缓解与收尾

```
[dlg_03_08]
Lendro：（手从枪套上挪开，有些不自然地摸了摸下巴。）
Lendro：……是。这地方太空了。刚才好像……有点太安静了。
（他端起桌上的酒杯/水杯，一口喝完。动作掩饰着刚才的失态。）
Lendro：我得回去了。局里还有几份报告没写。
（他站起来，抓起帽子。动作比刚来时快了很多。）
老麦：（看着他，没点破任何事。）
老麦：伞拿好。
Lendro：嗯。
（他走向门口，手握住门把的时候，停了一下，但没回头。）
Lendro：那疯子说得不对。时间要是真停了……根本受不了。
```

```

木头吧台上，留下了几道因为极度用力而抓出来的、很浅的指甲划痕。
老麦拿过抹布，用力擦了擦。
```
