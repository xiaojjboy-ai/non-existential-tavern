## 对话层

## 黑猫前奏

```
[dlg_02_cat]
（窗外一声轻微的刮擦。小猫的爪子正好踩在了昨晚那枚残留的泥污爪印上。）
（老麦抬头——一只黑色小猫蹲在窗台上。戴着一顶漂亮的深紫色魔法帽，帽檐歪斜，对她来说大了两号，却有种古怪的合适。脖子上挂着一块水晶吊坠，在夕照里微微泛着幽光。）
（不是野猫——毛色干净，眼神太灵。它没叫。没蹭。只是坐着。看着老麦。那眼神像极了某种考核前的督察员。）
老麦：（看着猫。声音不高。）
老麦：这年头动物进来也不定会花钱。
（猫没动。尾巴尖轻轻晃了一下——吊坠跟着晃了一下，折射了一小片光斑在吧台上。）
老麦：小家伙，我不招待童工和宠物。你自己找地方待着吧。
（猫像是听懂了一样——尾巴尖又晃了一下。沿着吧台走了一圈，跳下去，在靠窗的桌子底下趴下。隐藏在阴影里。开始——观察。）
```

## 安尼尔入场

```
[dlg_02_01]
（风铃声。门开了。进来的是一个人——体型偏胖，发际线稀疏。穿着一件不大考究的西服外套，但从领口能看出一截吊带工服的边缘。步子很沉。）
（老麦的鼻子动了动。比眼睛先一步做了判断。）
（三种味道——车间深处的重机油味。新贵族的廉价薰衣草香水。劣质烟草。混在一起，极其矛盾。）
老麦：（心里想——想跟上流人士打交道的车间老鼠。他以前见得多了。）
（安尼尔走到吧台前。站了一秒。没急着坐。似乎在评估这个地方够不够“档次”，或者够不够“安全”。）
安尼尔：……开着？
老麦：（点头。没多说话。）
安尼尔：（点头。像是确认了一个不需要回答的问题。）
```

### 选项 ch_02_01：老麦怎么接

```
[dlg_02_02a]
（老麦拿起手边的杯子。开始擦。没看安尼尔。动作利索——公事公办。）
老麦：坐。
（没说"随便坐"。没给酒单。不想多交流。这是个会走的客人，不需要他多花力气。）
安尼尔：（注意到了老麦的冷淡。拉凳子坐下。动作很重——凳脚刮过地板，发出一声刺耳的摩擦。）

[dlg_02_02b]
（老麦拿起杯子。开始擦。擦了一下。停了一下。多看了安尼尔一眼。）
（那双眼睛里没有暴发户的傲慢，也没有真正的自信。安尼尔只是坐在那里。手指在吧台上轻轻敲了一下——那是极度疲惫的敲击。）
老麦：（停了一秒。）
老麦：……坐吧。
（语气比计划中轻了一点。）

[dlg_02_02c]
（老麦没拿杯子。手撑在吧台上。）
老麦：你袖子上有机油。
安尼尔：（愣了一下。低头猛地看向自己的左袖——那里有一块硬币大小的新油渍，还没干。他本能地用右手遮了一下，又觉得这动作太蠢，硬生生放下了。）
安尼尔：……是。今天去下面车间转了一天。
老麦：（停了一秒。没揭穿那是深层机油的味道，绝不是随便“转转”能染上的。）
老麦：坐吧。
```

## 落座要求

```
[dlg_02_03]
（安尼尔坐下。凳子承重地咯吱了一下。他的手指在吧台上摊开——粗短、有力，指甲缝里有怎么洗都洗不掉的黑色机油垢。）
安尼尔：有酒单吗？来点……来点硬的。
（他本来想说点体面的话，但太累了，话到嘴边变成了最本能的需求。）
```

## 调酒反馈

```
[dlg_02_drink_correct]
（老麦没拿酒单，直接压了一大杯带厚厚白色泡沫的生啤过去。冰凉，冒着白气。）
（这是劳动阶级最稳妥的口粮。）
安尼尔：（看着那杯啤酒。愣了一下。他本想拒绝这种不符合他那身西装的饮料，但喉咙咽了一下。）
安尼尔：……你怎么知道我渴了。
（他端起杯子，咕咚咕咚灌了半杯。白色的泡沫沾在他的胡茬上。紧绷了一天的腰背瞬间塌了下来。）

[dlg_02_drink_wrong_whisky]
（老麦倒了一杯昂贵的威士忌，加了冰球。推过去。这是匹配他那身西装的选择。）
安尼尔：（盯着那颗晶莹剔透的冰球。手指搓了搓。）
安尼尔：……太烈了。这玩意儿喝完，我明天的标书就废了。
（他把杯子推回去一点。）
安尼尔：换一杯啤酒。起沫的那种。

[dlg_02_drink_wrong_white]
（老麦倒了一杯清透的白酒，推过去。）
安尼尔：（闻了一下，直摇头。）
安尼尔：酒精味太冲，烧胃。车间里那些酒鬼才喜欢这个。
安尼尔：给我换一杯啤酒吧。凉的。

[dlg_02_drink_wrong_other]
（老麦推过去一杯五颜六色的调酒。）
安尼尔：（眉头直接皱成了川字。）
安尼尔：……你们是不是觉得我是来参加什么化妆舞会的？
安尼尔：拿走拿走。来杯啤酒。要大杯的。
```

## 安静的间隙

```
[dlg_02_04]
（啤酒上来后，安尼尔解开西服扣子，从内侧口袋里掏出一卷旧纸——被折了很多次，边角磨毛了。摊在吧台上。）
（他没解释这是什么。老麦也没问。）
（安静。只有老麦擦杯子的声音，和安尼尔拿起一截磨短了的铅笔，在纸面划过的沙沙声。）
（安尼尔盯着图纸上某个点，没动笔。手指在吧台上敲了一下——不是不耐烦，是在计算。老麦认识这个动作——车间里的老钳工在等机器停转的时候也这样敲。）
（窗户阴影里，黑猫的尾巴轻轻晃了一下。它讨厌这股机油味，但它对纸上的图画似乎有些好奇。）
```

### 选项 ch_02_02：老麦是否打破沉默

```
[dlg_02_05a]
（老麦继续擦杯子。没说话。安静是今晚的默认状态。）
（安尼尔也没有打破它的意思。两个人隔着吧台——一个看图纸，一个擦杯子。这是一种不需要交际应酬的罕见舒适区。）
（安尼尔端起啤酒又喝了一口。铅笔继续在纸上走。）

[dlg_02_05b]
（老麦擦完一个杯子。放到架子上。然后——多拿了一个杯子。开始擦。为了有理由多站一会儿。）
老麦：……那是图纸？
安尼尔：（抬头。没想到老麦开了口。手指在纸上停了一下。）
安尼尔：……嗯。
（沉默。但他看了看手里的破铅笔，还是没忍住开口。）
安尼尔：一个不可能的东西。呵呵。
（他把啤酒端起来喝了一口。语气里有自嘲，但也有一丝他自己都没察觉的骄傲。老麦没问第二句——他知道什么时候该闭嘴。）

[dlg_02_05c]
（老麦擦杯子。看安尼尔低头画图的侧脸。啤酒在旁边喝了两口，泡沫在杯沿上渐渐塌下去。）
老麦：下班过来的？
安尼尔：（没抬头。）
安尼尔：……没下班。从车间出来，顺路。
老麦：车间？
安尼尔：（铅笔停了一下。抬头。表情立刻换上了一副假模假式的体面。）
安尼尔：蒸汽列车制造中心。我是那里的……主管。
老麦：（手没停。看了他一眼。）
老麦：不像。
安尼尔：（愣了一下，防御姿态瞬间拉满。）
老麦：主管不会是这个味道。
（安尼尔低头看了自己的左袖。那坨机油。然后——他肩膀垮了下来，防御被这句陈述句轻易击碎了。嘴角扯了一下，那是被戳穿后的释然。）
安尼尔：……是。每天都得下一线。锅炉那边的新人，不盯着不行。他们根本不懂得尊重机器。
```

## 收尾

```
[dlg_02_06]
（啤酒喝到一半。安尼尔看了看窗外——深蓝已经变成墨黑。第一颗星出来了。）
安尼尔：该走了。还有些图纸今晚要改完。
（把铅笔插回口袋。卷图纸的动作很慢——一层层叠回去，每一层都贴边对齐。不是整理文件的习惯，是组装精密仪器的手艺。）
（安尼尔站起来。手指撑了一下吧台——坐久了腿有点僵。）
安尼尔：……这地方挺安静的。
（不是"好"，是"安静"。对他来说，不需要强装上流社会的安静，就是最好的。）
```

### 选项 ch_02_03：告别

```
[dlg_02_farewell_a]
老麦：（收走空杯。杯底还有小半圈泡印。）
老麦：还开着。
安尼尔：（点了一下头。把西服扣子扣上一颗。走到门口。）
（黑猫的耳朵在阴影里转了一下，跟踪他每一步。）
（门关上。风铃响了一声。）

[dlg_02_farewell_b]
老麦：（没拿走杯子。让他自己喝完的已经是个信号。）
老麦：下次来。
安尼尔：（停了一下。像是没想到。然后点了一下头。）
安尼尔：……嗯。
（推门出去。步子比进来时轻了——不是身体轻了，是心里的那件西装被脱掉了。）

[dlg_02_farewell_c]
（老麦什么都没说。把一块新的杯布放在吧台上——不是给他，但放在他能看见的位置。和赶人走不一样。）
安尼尔：（看了一眼那块布。又看了一眼老麦。）
安尼尔：……走了。
（安尼尔推门出去。老麦等门关上后才拿起那块新布。继续擦杯子。）
```

## 猫咪互动 — 酒馆的新客人

```
[dlg_02_cat_bar]
（门关上。风铃余响散尽。安静了几秒。）
（黑猫从窗下阴影里走出来——帽子差点掉了，它甩了一下头扶正。踱到吧台前——安尼尔刚才坐过的位置。低头嗅了嗅凳子腿。打了个极度嫌弃的喷嚏。）
（它讨厌机油味。）
（然后它跳上了吧台。动作很轻，像一片羽毛落地。坐在老麦面前。魔法帽歪得更厉害了——左边帽檐快要遮住整只眼睛。脖子上的水晶吊坠在吧台灯下泛着幽紫色微光。）
（安尼尔的杯子还在吧台上。猫低头闻了闻——耳朵向后压了一下。也不喜欢啤酒。）
老麦：（看着猫。擦杯子的手没停。）
老麦：……你比那个人还挑剔。
（猫没理他。尾巴扫过安尼尔摊过图纸的位置，然后在吧台正中间坐下来。两只前爪并拢。它看着老麦。视线从老麦的手，移动到老麦的脸。像在等他做决定。）
```

### 选项 ch_02_cat：怎么对待这只猫

```
[dlg_02_cat_pet]
（老麦伸出手。手背朝上——车间里学来的规矩，先让对方检查你手里没有危险。）
（猫低头。鼻尖碰了一下老麦的指节。凉的。湿的。帽子往前滑——老麦的另一只手条件反射地接了一下帽檐，没让它掉。猫没躲。然后——它把头顶蹭进了老麦的掌心。）
（不是撒娇。是批准。像是这只魔物签了一份只有他们两个懂的入住契约。）
（老麦的手指轻轻挠过猫的耳后——那里有一小撮白毛，藏在紫色里。猫的眼睛眯起来。水晶吊坠贴着老麦的手腕——冰凉的。喉咙里发出一声极低的咕噜。）
老麦：（嘴角动了一下。极淡。）
老麦：……行了。帽子戴好。
（猫没动。他把手收回来。）

[dlg_02_cat_beer]
（老麦把安尼尔剩的啤酒杯推过去一点。杯底那浅浅一圈——刚好够猫舔一口。）
（猫低头闻了闻。抬头看老麦。帽檐下面那双眼睛里有一种"你在侮辱我的品味"的极度嫌弃。）
老麦：（不为所动。）
老麦：不喝算了。
（猫的尾巴不满地甩了一下——吊坠链子跟着轻轻响了一声。但它竟然真的低下头——快速舔了一口。然后立刻抬头，伸出舌头，抖了抖毛。）
（老麦没忍住，嘴角动了一下——极淡。猫恶狠狠地瞪了他一眼。帽子歪得只剩一只眼睛看着他。但没跳下去。依然霸占着吧台。） 
```

## 猫的回应

```
[dlg_02_cat_result]
（猫在吧台上踱了两步。帽子歪斜，吊坠拖在木头上划出一道细小的反光。它挑了个位置——正是安尼尔放过图纸的那块空处。转了两圈。蜷下来。）
（一只爪子搭在老麦擦杯子的手旁边。肉垫是凉的。没有伸出指甲。也没有碰到他。）
（老麦停了一下。没抽手。过了两秒，猫把爪子收回去，塞进肚子下面——顺便把发光的吊坠也压在肚子底下。闭上眼。）
老麦：你确实比刚才那个客人好伺候。
（声音很低。像是说给自己听的。）
```

## 旁白

```
[narr_02_01]
老麦看了一眼墙上的钟。离夜班结束还早。他拿起杯子——继续擦。
猫的耳朵跟着擦杯子的沙沙声转了半圈。帽子彻底滑下来，盖住了半边脸。
然后它不动了。在吧台上睡着了。
尾巴盖住了鼻子，刚好遮住耳后那撮白毛。吊坠从肚子底下滑出来——幽光暗了，但并没有熄灭。
酒馆迎来了它的第二位常客，或者说——真正的“房客”。
```
