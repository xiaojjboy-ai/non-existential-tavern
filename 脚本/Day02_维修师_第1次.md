## 对话层

## 黑猫前奏

```
[dlg_02_cat]
（窗外一声轻微的刮擦。老麦抬头——一只黑色小猫蹲在窗台上。不是野猫——毛色干净，眼神太灵。它没叫。没蹭。只是坐着。看着老麦。）
（老麦放下杯子。没赶，也没招手。一人一猫对视了三秒。猫的眼睛没有猫咪的好奇——那是一种审视。像督察在检查这间酒馆是否够格。）
老麦：（看着猫。声音不高。）
老麦：这年头客人来也不定会花钱。
（猫没动。）
老麦：小客人我就不招待你了。你自己找地方呆着吧。
（猫像是听懂了一样——尾巴尖轻轻晃了一下。然后沿着吧台走了一圈，跳下去，在靠窗的桌子趴下。正好在最后一缕夕照里。收起爪子。开始——观察。）
```

## 安尼尔入场

```
[dlg_02_01]
（风铃声。门开了。进来的是一个人——体型偏胖，发际线稀疏。穿着一件不大考究的西服外套，里面露出一截吊带工服的边缘。步子不快。像是走了一天的路。）
（老麦的鼻子动了动。比眼睛先一步做了判断。）
（三种味道——车间的机油。新贵族的薰衣草香水。烟草。混在一起，比任何自我介绍都准确。）
老麦：（心里想——跟车间打交道的上流人士。以前的工头也是这个味道。）
（安尼尔走到吧台前。站了一秒。没急着坐。）
安尼尔：……开着？
老麦：（点头。没多说话。）
安尼尔：（点头。像是确认了一个不需要回答的问题。）
```

### 选项 ch_02_01：老麦怎么接

```
[dlg_02_02a]
（老麦拿起手边的杯子。开始擦。没看安尼尔。动作利索——公事公办。）
老麦：坐吧。
（没说"随便坐"。没给酒单。不想多交流。这是个会走的客人，不需要他多花力气。）
安尼尔：（注意到了老麦的冷淡。没说什么。拉凳子坐下。动作很重——凳脚刮过地板。）

[dlg_02_02b]
（老麦拿起杯子。开始擦。擦了一下。停了一下。多看了安尼尔一眼。）
（那双眼睛里没有以前上司的傲慢。安尼尔只是坐在那里。手指在吧台上轻轻敲了一下——累的。不是不耐烦。）
老麦：（停了一秒。）
老麦：……坐吧。
（语气比计划中轻了一点。只是一点。）

[dlg_02_02c]
（老麦没拿杯子。手撑在吧台上。）
老麦：你身上有机油味。
安尼尔：（愣了一下。低头看了看自己的袖子——袖口有一块新的油渍，还没干。他自己都没注意到。）
安尼尔：……是。今天在车间待了一天。
老麦：（停了一秒。不知道这句是赞赏还是试探。）
老麦：坐吧。
```

## 落座点单

```
[dlg_02_03]
（安尼尔坐下。凳子承重地咯吱了一下。他的手指在吧台上摊开——粗短、有力，指甲缝里有洗不掉的黑色。）
安尼尔：啤酒。就啤酒。
（不是问有什么选择。是自己已经做好了决定。）
（老麦转身拿杯子。安尼尔解开西服扣子，从内侧口袋里掏出一卷旧纸——被折了很多次，边角磨毛了。摊在吧台上。）
（他没解释这是什么。老麦也没问。）
（老麦倒啤酒。满杯，泡沫刚好到杯口。推过去。）
安尼尔：（端起来喝了一口。然后低头看图纸。拿起笔——不是钢笔，是一截磨短了的铅笔——在上面改了点什么。）
```

## 安静的间隙

```
[dlg_02_04]
（安静。只有老麦擦杯子的声音，和安尼尔铅笔划过纸面的沙沙声。这两种声音混在一起，互不干扰。）
（安尼尔喝了一口啤酒。盯着图纸上某个点，没动笔。手指在吧台上敲了一下——不是不耐烦，是在想。老麦认识这个动作——车间里的工人在等机器停下来的时候也这样敲。）
（窗外，黑猫的尾巴在夕照里晃了一下。）
（安尼尔没注意到猫。老麦注意到了。）
```

### 选项 ch_02_02：老麦是否打破沉默

```
[dlg_02_05a]
（老麦继续擦杯子。没说话。安静是今晚的默认状态。）
（安尼尔也没有打破它的意思。两个人隔着吧台——一个在看图纸，一个在擦杯子。不是尴尬的沉默。是那种不需要说话也可以各干各的不说话。）
（安尼尔端起啤酒又喝了一口。泡沫在下唇留下了一道白。他没擦。）
（铅笔继续在纸上走。）

[dlg_02_05b]
（老麦擦完一个杯子。放到架子上。然后——多拿了一个杯子。开始擦。）
（为了多站一会儿。）
老麦：……那是图纸？
安尼尔：（抬头。没想到老麦开了口。手指在纸上停了一下。）
安尼尔：……嗯。
（沉默。）
安尼尔：一个不可能的东西。呵呵。
（把啤酒端起来喝了一口。没说是什么。老麦没问第二句。但擦杯子的手停了一下——"不可能的东西"这句话，他在别的地方听过。）

[dlg_02_05c]
（老麦擦杯子。看安尼尔低头画图的侧脸。啤酒在旁边——喝了两口。泡还在杯沿上渐渐塌下去。）
老麦：下班过来的？
安尼尔：（没抬头。）
安尼尔：……没下班。从车间出来，顺路。
老麦：车间？
安尼尔：（铅笔停了一下。抬头。）
安尼尔：蒸汽列车制造中心。我是那里的——
（顿了一下。不太想说职位。）
安尼尔：……管事的。
老麦：（手没停。）
老麦：不像。
安尼尔：（愣了一下。）
老麦：管事的不会这个味道。
（安尼尔低头看了自己的袖子。那坨机油。然后——嘴角动了一下。不是笑。是"被你发现了"。）
安尼尔：……每天都下一线。锅炉那边的新人——不盯着不行。
（老麦没接话。但擦杯子的节奏变了一点点。不是前任工头的节奏了。）
```

## 收尾

```
[dlg_02_06]
（啤酒喝到一半。安尼尔看了看窗外——深蓝已经变成墨黑。第一颗星出来了。）
安尼尔：该走了。还有些图纸今晚要改完。
（把铅笔插回口袋。卷图纸的动作很慢——一层层叠回去，每一层都贴边对齐。不是整理文件的手艺，是组精密零件的习惯。）
（安尼尔站起来。手指撑了一下吧台——坐久了腿有点僵。）
安尼尔：……这地方安静。
（不是"好"。是"安静"。对他来说是一回事。）
```

### 选项 ch_02_03：告别

```
[dlg_02_farewell_a]
老麦：（收走空杯。杯底还有小半圈泡印。）
老麦：还开着。
安尼尔：（点了一下头。把西服扣子扣上一颗。走到门口。）
（黑猫的耳朵转了一下。跟踪他每一步。然后收回。）
（门关上。风铃响了一声。）

[dlg_02_farewell_b]
老麦：（没拿走杯子。让他自己喝完的已经是个信号。）
老麦：下次来。
安尼尔：（停了一下。像是没想到。然后点了一下头。）
安尼尔：……嗯。
（推门出去。步子比进来时轻了——不是身体轻了。是少了一件要防的事。）

[dlg_02_farewell_c]
（老麦什么都没说。把一块新的杯布放在吧台上——不是给他。但放在他能看见的位置。和收杯子不一样。）
安尼尔：（看了一眼那块布。然后看了看老麦。老麦在擦杯子——没看他。）
安尼尔：……走了。
（安尼尔推门出去。老麦等门关上后才拿起那块新布。继续擦杯子。）
```

## 猫咪互动 — 酒馆的新客人

```
[dlg_02_cat_bar]
（门关上。风铃余响散尽。安静了几秒。）
（黑猫从窗台跳下来。踱到吧台前——安尼尔刚才坐过的位置。低头嗅了嗅凳子腿。打了个喷嚏。）
（然后它跳上了吧台。动作很轻，几乎没有声音。坐在老麦面前。看着老麦。眼神还是那种审视——但多了一点别的。也许是好奇。）
（安尼尔的杯子还在吧台上。杯底剩了浅浅的一圈啤酒。猫低头闻了闻——耳朵向后压了一下。不喜欢这个味道。）
老麦：（看着猫。擦杯子的手没停。）
老麦：……你也不喝啤酒？
（猫没理他。尾巴扫过安尼尔摊过图纸的位置，扫过那块新杯布。然后在吧台正中间坐下来。两只前爪并拢。像在等什么。）
老麦：（放下杯子。手在吧台上空着——没拿东西。他看着猫。猫看着他。）
（一人一猫。中间的空气可以伸手，也可以不伸手。）
```

### 选项 ch_02_cat：怎么对待这只猫

```
[dlg_02_cat_pet]
（老麦伸出手。手背朝上——先让猫闻。手指有啤酒花的味道，也有擦杯布的味道。）
（猫低头。鼻尖碰了一下老麦的指节。凉的。湿的。然后——它把头顶蹭进了老麦的掌心。）
（不是撒娇。是批准。像是签了一份只有他们两个懂的契约。）
（老麦没说话。手指轻轻挠过猫的耳后——那里有一小撮白毛，藏在黑色里，不仔细看看不见。猫的眼睛眯起来。喉咙里发出一声极低的咕噜——像是机器启动前的那一下震颤。）
老麦：（嘴角动了一下。极淡。）
老麦：……行了。下去吧。
（猫没动。他把手收回来。猫的耳朵追着他的手指转了一下。但没跟。）

[dlg_02_cat_beer]
（老麦把安尼尔剩的啤酒推过去。杯底那浅浅一圈——刚好够猫舔一口。）
（猫低头闻了闻。然后抬头看老麦。眼神里有一种"你在逗我"的意思。）
老麦：（不为所动。）
老麦：不喝算了。
（猫的尾巴不满地甩了一下。但还是低下头——舔了一口。然后立刻抬头，伸出舌头，一脸"这什么东西"的嫌弃表情。）
（老麦嘴角动了一下——极淡。猫瞪了他一眼。但没跳下去。还是在吧台上坐着。） 
```

## 猫的回应

```
[dlg_02_cat_result]
（猫在吧台上踱了两步。挑了个位置——安尼尔放过图纸的那块空处。转了两圈。蜷下来。）
（一只爪子搭在老麦擦杯子的手旁边。肉垫是凉的。没有伸出指甲。也没有碰到他。）
（老麦停了一下。没抽手。过了两秒，猫把爪子收回去，塞进肚子下面。闭上眼。）
老麦：你比刚才那个客人好伺候。
（声音很低。像是说给自己听的。）
```
