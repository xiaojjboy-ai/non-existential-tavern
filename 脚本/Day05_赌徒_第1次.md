## 对话层

## 入场

```
[dlg_05_01]
（门被极其小心地推开了一条缝。一只布满血丝的眼睛先从门缝里扫视了一圈酒馆，确认没有别的客人后，才闪身进来。）
Aletor：（动作轻得像猫，但呼吸很重，显然是在极力压抑。他身上混杂着废土泥浆和昂贵的魔法布料气味。右手缩在袖子里，似乎握着什么东西。）
Aletor：（他没有立刻走向吧台，而是背靠着墙，死死盯着老麦。）
Aletor：……门能上锁吗？我不喜欢背后有风。
```

### 选项 ch_05_01：老麦怎么接

```
[dlg_05_02a]
老麦：（停下擦杯子的手。）
老麦：开门做生意。锁门就成黑店了。
Aletor：（眯起眼睛，手指在袖子里动了一下。他判断了一下老麦的威胁程度，最终放弃了争执。）
Aletor：……行。如果有看起来不像人的东西进来，记得先喊一声。

[dlg_05_02b]
老麦：（没看门，继续擦杯子。）
老麦：外面有人追你？
Aletor：（听到“人”这个词，他的肌肉下意识地紧绷了一下，但马上被他用干笑掩饰过去。）
Aletor：哈。仇家而已。欠了点还不上的烂账。

[dlg_05_02c]
老麦：（放下杯子，看了看他泥泞的鞋底，又看了看他紧绷的肩膀。）
老麦：最里面那个座位在死角，门外看不见。去那坐着，别碰倒我其他客人的椅子。
Aletor：（顺着老麦的视线看了看那个角落。这是一个绝佳的防御位置。）
Aletor：……你倒是个懂行的。
（他像一只贴着墙根的耗子，迅速滑进了那个角落的位置。）
```

## 落座与要求

```
[dlg_05_03]
Aletor：（缩在角落的高脚凳上。此时他才从怀里摸出一块古铜色的怀表，看了一眼，迅速合上。右手始终没离开袖子里露出一截的紫色法杖。）
Aletor：酒。最烈的。
Aletor：能让我立刻麻痹半个小时、但又不会真的睡死过去的那种。快点。
```

## 调酒反馈

```
[dlg_05_drink_correct]
（老麦倒了一杯伏特加，不加冰，不兑水。推过去。）
Aletor：（小心翼翼地闻了一下，然后一口灌下大半杯。）
Aletor：咳——够劲。
（酒精让他的紧绷感稍微缓解了一丝，他终于敢把左手从怀表上拿开一会儿。）

[dlg_05_drink_wrong_whisky]
（老麦倒了一杯威士忌。推过去。）
Aletor：（喝了一小口，皱了皱眉。）
Aletor：不够。这玩意只能骗骗小屁孩。
Aletor：算了，凑合吧。现在也不是挑剔的时候。
（他又摸了摸怀表。）

[dlg_05_drink_wrong_beer]
（老麦推过去一杯冰啤酒。）
Aletor：（看了一眼，脸色阴沉下来。）
Aletor：你觉得我像是有闲工夫在这儿品尝麦芽香气的人吗？
Aletor：我要的是能麻痹神经的药！不是水！

[dlg_05_drink_wrong_water]
（老麦倒了一杯水。对处于极度戒备的人，酒精可能会误事。）
Aletor：（死死盯着那杯水，突然神经质地低声笑了一下。）
Aletor：水……哈哈。你觉得我需要清醒？
Aletor：我就是太清醒了，才会被那个无时无刻不在倒计时的声音逼疯。
```

## 开启话题 — 含糊其辞的“讨债人”

```
[dlg_05_04]
Aletor：（烈酒下肚，他稍微放松了一点戒备，但目光依然时不时瞟向门口。）
Aletor：你这儿平时，常有那种……奇怪的客人来吗？
Aletor：我的意思是，不留脚印的，或者，周围空气比冰块还冷的。
（他用手指无意识地敲击着怀表表面，声音很低，像是在自言自语。）
Aletor：我跑过很多绝境。时间之峡、虚空废墟……什么怪物我没见过？但这次这个“讨债的”，简直是个阴魂不散的疯子。
```

### 选项 ch_05_02

```
[dlg_05_05a]
老麦：（擦杯子。语气平淡。）
老麦：既然欠了账，还了就是。
Aletor：（像是听到了什么极其荒谬的笑话，脸皮抽动了一下。）
Aletor：还？拿什么还？命只有一条！
Aletor：他不收金币，不收魔法晶石……他只要我停下。只要我停下，那一天就是我的死期。

[dlg_05_05b]
老麦：（看着他敲击怀表的手。）
老麦：跑得掉吗？
Aletor：（手突然停住，死死护住怀表。）
Aletor：只要我有这个……只要我还能跳出他的规则一秒钟，他就抓不到我！
Aletor：我可是这个世界上最伟大的探……哼，过去的事没必要提了。总之，我能赢。

[dlg_05_05c]
老麦：（擦完一个杯子，放好。）
老麦：听起来你惹了不该惹的东西。
Aletor：（眼神暗了下去，咬着牙。）
Aletor：是规则惹了我。我只是比普通人多跨过了几次那道墙而已……他们就派了条最冷的狗来咬我。
```

## 核心对话 — 怀表与法杖

```
[dlg_05_06]
Aletor：（他小心翼翼地把紫色法杖放在吧台上，但手一寸也没离开。法杖上的纹路极其古老复杂。）
Aletor：这是我花了大代价从学院弄来的。那群只知道坐在塔里看书的蠢货，根本不知道这玩意能在外面救多少次命。
Aletor：虽然是个天才女学员的残次品，限制多得要命……但配合我自己的“小发现”——
（他摸了摸怀表，没有说出时停的秘密，探险家的谨慎让他适可而止。）
Aletor：算了，跟你说这些干什么。你只要知道，我还活着，而且我还会继续活下去。
```

### 选项 ch_05_03

```
[dlg_05_07a]
老麦：（拿走空酒杯。没评价他的恐惧。）
老麦：这酒你打算付钱吗？
Aletor：（愣住，然后低声笑了。这种极其世俗的对话让他感到安全。）
Aletor：放心。我不欠活人的钱。
（丢下一枚成色极好的古币。）

[dlg_05_07b]
（老麦把那杯倒给猫喝的魔法药酒的瓶子，稍微往旁边推了推，免得被这个神经质的男人碰倒。）
Aletor：（锐利的目光立刻扫过那个药酒瓶，职业习惯让他迅速评估了一下药剂成分，然后失去了兴趣。）
Aletor：基础劣质魔力药剂。这种东西连给我恢复体力都不够。

[dlg_05_07c]
（老麦什么也没做，只是站在那儿，作为一个安静的旁听者。）
Aletor：（看了老麦一会儿。他很满意这个酒保的识趣。）
Aletor：你是个好酒保。话少，活得长。
```

## 收尾与探险家的阅历 (猫的共振)

```
[dlg_05_08]
（就在这时，放在吧台上的紫色传送法杖突然发出微弱的“嗡”鸣声。紫光轻微闪烁。）
Aletor：（探险家的本能让他瞬间弹起，一把抓起法杖。但并没有盲目传送，而是迅速寻找魔力源头。）
Aletor：共鸣？附近有同源魔力？！
（他顺着法杖微光的方向看去——趴在窗台上的黑猫正站起身。猫脖子上的水晶吊坠和法杖产生了轻微的感应。）
Aletor：（眯起眼睛，戒备地盯着那只猫，眼底流露出极其老辣的观察力。）
Aletor：……魔法生物？
Aletor：不……骨骼和肌肉完全是一只普通的流浪猫。但这个魔力回路是硬塞进去的。后天改造？
Aletor：（他厌恶地皱起眉头。）哪个疯子会把一个高阶灵魂强行塞进一只连数字都数不清的猫里？这比杀人还恶心。
```

## 退场

```
[narr_05_01]
法杖的嗡鸣声稍微变大了一点。
在黑猫的背后，空气微微扭曲，隐约闪过一个小女孩穿着宽大魔法袍的虚影。
虚影极度模糊，只存在了不到半秒，便像信号不良的画面一样溃散了。
但这一闪而过的异常，足以触动一个常年在生死边缘游走的探险家的神经。
“见鬼……这地方也不干净。”
Aletor 没有任何犹豫，将怀表塞进贴胸的口袋，握紧法杖，压低帽檐。
他没有夺门而出，而是像来时一样，轻手轻脚、却以极快的速度滑出了酒馆。
门被无声地关上。
老麦收起吧台上的古币。窗台上的黑猫甩了甩脑袋，似乎对刚才那半秒钟的虚影毫无察觉，又重新趴下睡觉了。
但这轻微的共振，已经在猫的躯壳里种下了一颗即将苏醒的种子。
```
