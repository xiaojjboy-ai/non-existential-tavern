## 对话层

## 入场

```
[dlg_06_01]
（门被推开。Lendro 走进来。今晚他的动作比前两次都要慢一些。）
（风衣上的雨水顺着下摆往下滴。他没有立刻开口讲那个招牌式的开场笑话，而是摘下帽子，弹了弹水珠，才走向吧台。）
Lendro：呼……见鬼的天气。
（他在老位置坐下。看了一眼窗台上的黑猫，又看了一眼老麦。）
Lendro：今天码头那边出了点乱子。几个走私犯为了两箱假酒，在仓库里拔了枪。
Lendro：我带队进去的时候，满地的血……和碎玻璃。
（他突然停顿了。前两次来访，他绝不会在这个时候停顿，他会立刻接一个冷笑话把血腥味盖过去。但这次，他停住了。）
```

### 选项 ch_06_01：老麦怎么接

```
[dlg_06_02a]
老麦：（没接话，只是默默地把一块干净的干毛巾推到他面前。）
Lendro：（看着毛巾，愣了一下。然后拿起来擦了擦脸上的雨水。）
Lendro：谢了。那玻璃渣子混着血，踩上去嘎吱嘎吱响……算了。老规矩。

[dlg_06_02b]
老麦：（手上的动作没停。）
老麦：然后呢？
Lendro：（脸上的肌肉抽动了一下，似乎在想要不要编个笑话。）
Lendro：然后？然后他们就被逮了呗。还能有什么然后……给我来杯老规矩。

[dlg_06_02c]
老麦：（看了他一眼。）
老麦：听起来不像是个好笑的故事。
Lendro：（干笑了一声，但这次眼睛完全没有笑意。）
Lendro：一点也不好笑。特别是当你闻到那种味道的时候……老规矩，麦克。
```

## 调酒反馈

```
[dlg_06_drink_correct]
（老麦倒了一杯威士忌。推过去。）
Lendro：（拿起杯子，喝了一大口。长长地舒了一口气。）
Lendro：就是这个。能把喉咙里的铁锈味洗下去。

[dlg_06_drink_wrong_vodka]
（老麦倒了一杯伏特加。推过去。）
Lendro：（喝了一口，被呛得咳嗽起来。）
Lendro：咳咳……太烈了。今天不需要烧脑子，我需要点能慢慢咽下去的。不过算了。

[dlg_06_drink_wrong_beer]
（老麦推过去一杯啤酒。）
Lendro：（看了一眼，没碰。）
Lendro：太淡了，麦克。这压不住那种恶心的感觉。给我换威士忌。

[dlg_06_drink_wrong_water]
（老麦倒了一杯水。）
Lendro：（盯着水杯看了几秒，苦笑。）
Lendro：如果水能洗干净脑子里的东西，我就不会在这个点来找你了。换威士忌吧。
```

## 核心对话 — 允许安静

```
[dlg_06_03]
Lendro：（喝了半杯威士忌。他转动着手里的玻璃杯，盯着里面的琥珀色液体。）
Lendro：你知道最可笑的是什么吗？那个开枪的走私犯，被我按在地上的时候，竟然在哭。
Lendro：他说他不想开枪的。他说太黑了，太安静了，他害怕。
（Lendro 的声音越来越低。他似乎不再是在跟老麦说话，而是在跟杯子里的自己说话。）
Lendro：太安静了……是啊，仓库里一旦静下来，确实会让人产生幻觉。
（他张了张嘴，似乎习惯性地想讲个笑话来圆场。）
Lendro：就像……就像……
（但他没说出来。他放弃了。他闭上了嘴。酒馆里陷入了长达十几秒的完全沉默。）
（只有留声机里微弱的爵士乐，和窗外的雨声。）
```

### 选项 ch_06_02：老麦的擦杯子节奏

```
[dlg_06_04a]
（老麦拿起一块纯棉的白布，不紧不慢地擦拭着一个玻璃杯。）
（擦拭的声音非常有规律。一下。一下。一下。）
Lendro：（在这极度规律的声音里，他紧绷的肩膀竟然奇迹般地放松了一点。他没有被这种安静逼疯。）
Lendro：……你这布哪买的？擦得挺干净。

[dlg_06_04b]
（老麦放下杯子，给自己也倒了一杯水。喝了一口。什么也没说。）
Lendro：（看着老麦喝水。老麦的存在本身就是一种“不需要说话也行”的证明。）
Lendro：……有时候我挺羡慕你的。每天晚上就面对这些不会说话的杯子。

[dlg_06_04c]
（老麦转身去整理身后的酒架，把几瓶空酒瓶放进回收箱。玻璃碰撞发出沉闷的低频声。）
Lendro：（这恰到好处的环境音填补了那块让他恐惧的绝对空白。）
Lendro：……这批空瓶子能卖几个钱？
```

## 退场

```
[narr_06_01]
Lendro 把剩下的半杯酒喝完。
他今晚说的废话比以前少了一半。
他站起身，扣好风衣。
“走了。”
老麦点点头。
如果是以前，Lendro 会在门口再讲一句烂笑话再推门。但今天，他只是摆了摆手，走进了雨里。
他第一次在这个酒馆里，允许了“安静”的存在。
风铃响过。酒馆打烊。
老麦关掉了大灯，只留下了吧台前的一盏小灯。
角落里，那个装满安尼尔药酒的粗糙玻璃瓶，正静静地散发着微光。
窗台上的黑猫站了起来。它戴着那顶偏大的深紫色魔法帽，脖子上的水晶吊坠死死对着那个酒瓶，发出同频的幽光。
该喂猫了。
```

### 打烊喂猫仪式

```
[dlg_06_feed_normal]
（老麦从柜台下拿出一盘干猫粮和一碟清水，推到窗台上。）
（黑猫闻了闻，嫌弃地偏过头，耳朵往后压了压，甚至发出了一小声不满的“嘶”。）
（它依然死死盯着角落里的药酒瓶。）
老麦：挑食可不是好习惯。
（黑猫没理他，重新趴下，用尾巴把自己卷起来。什么也没有发生。）

[dlg_06_feed_magic]
（老麦拿起角落里的那瓶维修师留下的粗糙药酒。拔掉软木塞，一股极其刺鼻但蕴含着庞大魔力的气息散开。）
（老麦在一个小木碟里倒了一点。推到窗台。）
（黑猫立刻凑了上来。它的舌头刚碰到液体，整个酒馆的气场瞬间就变了。）
（黑猫脖子上的紫水晶吊坠爆发出刺目的光芒！光芒如同实质般向四周蔓延，把墙壁上的影子拉得极长。）
（老麦下意识地抬手挡了一下眼睛。）
（光芒中心，猫的躯体上方，空气开始剧烈扭曲、重组。）
（那不再是 Day 05 赌徒来临时仅有半秒的虚影。这一次，借由药酒的庞大供能，那个虚影彻底稳固了下来。）
（一个穿着宽大、极不合身的学院法师袍的小女孩灵体，怯生生地站在窗台上。袍子的袖口卷了好几道，魔法帽歪歪扭扭地扣在头上。）
（她只有七八岁的样子，身体是半透明的紫色。）
（她低头看了看自己半透明的手，又看了一眼老麦。）
（小女孩：“……重。好重……”）
（她的声音空灵，带着极度的迷茫和怯懦。）
（仅仅维持了几秒钟，灵体就像耗尽了电量一样闪烁了一下，瞬间溃散回猫的体内。）
（黑猫摇晃了一下，倒在窗台上，发出了均匀的鼾声。它吃饱了。）
```

## 最终收尾

```
[narr_06_02]
老麦看着空了的木碟，又看了看睡死的黑猫。
“……看来明天得让安尼尔再带几瓶过来了。”
老麦收起木碟，锁上了酒馆的门。
在这个唯心与唯物交错的夜晚，有些东西，终于醒了。
```
